# Pie chart (composition)

`PieChart` is a ready-to-use composition for part-to-whole categorical distributions.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of data objects with name/value keys |
| `valueKey` | `string` | `'value'` | Key in each data object for the numeric value |
| `nameKey` | `string` | `'name'` | Key in each data object for the slice label |
| `colors` | `string[]` | theme colors | Array of fill colors for slices |
| `innerRadius` | `string \| number` | `0` | Inner radius (set > 0 for a donut effect) |
| `outerRadius` | `string \| number` | `'80%'` | Outer radius of the pie |
| `paddingAngle` | `number` | `2` | Gap in degrees between slices |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLabels` | `boolean` | `false` | Render percentage labels on slices |
| `animate` | `boolean` | `false` | Enable entry animation |
| `height` | `number` | `300` | Chart height in pixels |

## Variant — default

```tsx preview
import { PieChart } from '@geti-ai/charts';

const data = [
  { name: 'Images', value: 52 },
  { name: 'Videos', value: 18 },
  { name: 'Annotations', value: 22 },
  { name: 'Models', value: 8 },
];

export default function App() {
  return (
    <PieChart
      data={data}
      aria-label="Dataset storage share"
      height={320}
    />
  );
}
```

## Variant — labels on

```tsx preview
import { PieChart } from '@geti-ai/charts';

const data = [
  { name: 'Car', value: 4200 },
  { name: 'Person', value: 5600 },
  { name: 'Bicycle', value: 2100 },
  { name: 'Bus', value: 1400 },
];

export default function App() {
  return (
    <PieChart
      data={data}
      showLabels
      aria-label="Class distribution with labels"
      height={320}
    />
  );
}
```

## Variant — no legend, custom palette

```tsx preview
import { PieChart } from '@geti-ai/charts';

const data = [
  { name: 'Accepted', value: 71 },
  { name: 'Rejected', value: 19 },
  { name: 'Needs review', value: 10 },
];

export default function App() {
  return (
    <PieChart
      data={data}
      showLegend={false}
      colors={['#22c55e', '#ef4444', '#f59e0b']}
      aria-label="Review status split"
      height={300}
    />
  );
}
```

## Variant — train / validation / test palette

```tsx preview
import { PieChart, getDatasetSubsetColor } from '@geti-ai/charts';

const data = [
  { name: 'Train', value: 70 },
  { name: 'Validation', value: 20 },
  { name: 'Test', value: 10 },
];

export default function App() {
  return (
    <PieChart
      data={data}
      colors={[
        getDatasetSubsetColor('train')!,
        getDatasetSubsetColor('validation')!,
        getDatasetSubsetColor('test')!,
      ]}
      showLabels
      aria-label="Dataset subset split"
      height={320}
    />
  );
}
```
