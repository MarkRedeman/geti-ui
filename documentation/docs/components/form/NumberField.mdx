# NumberField

NumberField is a text input constrained to numeric values with built-in increment/decrement stepper buttons. Use it for threshold values, quantities, confidence scores, and any other numeric parameter input.

```tsx
import { NumberField } from '@geti-ui/ui';
```

- [React Spectrum NumberField ↗](https://react-spectrum.adobe.com/react-spectrum/NumberField.html)
- [Geti UI Theming](/components/theming)

## Examples

### Basic

```tsx preview
import { NumberField } from '@geti-ui/ui';

export default function App() {
  return <NumberField label="Quantity" defaultValue={1} />;
}
```

### With min/max and step

```tsx preview
import { NumberField } from '@geti-ui/ui';

export default function App() {
  return (
    <NumberField
      label="Confidence threshold (%)"
      defaultValue={50}
      minValue={0}
      maxValue={100}
      step={5}
    />
  );
}
```

### With format options (percentage)

```tsx preview
import { NumberField } from '@geti-ui/ui';

export default function App() {
  return (
    <NumberField
      label="Score"
      defaultValue={0.75}
      formatOptions={{ style: 'percent' }}
    />
  );
}
```

### Disabled

```tsx preview
import { NumberField } from '@geti-ui/ui';

export default function App() {
  return <NumberField label="Locked value" defaultValue={42} isDisabled />;
}
```

## Responsibility

`NumberField` is a thin wrapper around React Spectrum's `NumberField`. All props pass through unchanged. The wrapper provides a `@geti-ui/ui` import path and Geti theming.

## Variants and states

| State | Prop | Notes |
|---|---|---|
| Disabled | `isDisabled` | Non-interactive |
| Read-only | `isReadOnly` | Value visible but not editable |
| Invalid | `validationState="invalid"` | Shows error styling |
| Required | `isRequired` | Marks field as required in forms |

## Props (deep dive)

### Content

- `label`: visible field label.
- `description`: helper text below the field.
- `errorMessage`: error text when `validationState="invalid"`.

### Value

- `value`: controlled numeric value.
- `defaultValue`: uncontrolled initial value.
- `minValue` / `maxValue`: clamps the acceptable range.
- `step`: increment/decrement amount for stepper buttons and arrow keys.

### Format

- `formatOptions`: `Intl.NumberFormatOptions` - controls display format (currency, percentage, decimals).

### Events

- `onChange`: `(value: number) => void`.

### Accessibility

- always provide a `label`,
- `minValue` / `maxValue` are exposed as `aria-valuemin` / `aria-valuemax`.
