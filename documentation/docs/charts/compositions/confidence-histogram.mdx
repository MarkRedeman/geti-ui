# Confidence histogram (composition)

`ConfidenceHistogram` shows the distribution of prediction confidences and helps select thresholds.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `{ binStart: number; binEnd: number; count: number }[]` | — | Array of histogram bins with start, end, and count |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `animate` | `boolean` | `false` | Enable entry animation |
| `showFrequency` | `boolean` | `false` | Normalize bars to relative frequency (0–1) instead of raw counts |
| `threshold` | `number` | — | Draw a vertical reference line at this confidence value |
| `xAxisProps` | `XAxisProps` | — | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | — | Extra props forwarded to the Recharts `YAxis` |

## Variant — raw confidence counts

```tsx preview
import { ConfidenceHistogram } from '@geti-ai/charts';

const data = [
  { binStart: 0.0, binEnd: 0.1, count: 12 },
  { binStart: 0.1, binEnd: 0.2, count: 21 },
  { binStart: 0.2, binEnd: 0.3, count: 33 },
  { binStart: 0.3, binEnd: 0.4, count: 57 },
  { binStart: 0.4, binEnd: 0.5, count: 79 },
  { binStart: 0.5, binEnd: 0.6, count: 104 },
  { binStart: 0.6, binEnd: 0.7, count: 132 },
  { binStart: 0.7, binEnd: 0.8, count: 162 },
  { binStart: 0.8, binEnd: 0.9, count: 141 },
  { binStart: 0.9, binEnd: 1.0, count: 93 },
];

export default function App() {
  return (
    <ConfidenceHistogram
      data={data}
      aria-label="Prediction confidence histogram"
      height={320}
    />
  );
}
```

## Variant — normalized frequency with threshold line

```tsx preview
import { ConfidenceHistogram } from '@geti-ai/charts';

const data = [
  { binStart: 0.0, binEnd: 0.1, count: 12 },
  { binStart: 0.1, binEnd: 0.2, count: 21 },
  { binStart: 0.2, binEnd: 0.3, count: 33 },
  { binStart: 0.3, binEnd: 0.4, count: 57 },
  { binStart: 0.4, binEnd: 0.5, count: 79 },
  { binStart: 0.5, binEnd: 0.6, count: 104 },
  { binStart: 0.6, binEnd: 0.7, count: 132 },
  { binStart: 0.7, binEnd: 0.8, count: 162 },
  { binStart: 0.8, binEnd: 0.9, count: 141 },
  { binStart: 0.9, binEnd: 1.0, count: 93 },
];

export default function App() {
  return (
    <ConfidenceHistogram
      data={data}
      showFrequency
      threshold={0.62}
      yAxisProps={{ tickFormatter: (value) => `${Math.round(Number(value) * 100)}%` }}
      aria-label="Confidence histogram with normalized frequencies and threshold"
      height={320}
    />
  );
}
```
