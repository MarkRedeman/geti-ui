# CheckboxGroup

CheckboxGroup manages a set of related checkboxes with a shared label and a single value array. Use it when users can select any combination of options from a named list.

```tsx
import { Checkbox, CheckboxGroup } from '@geti-ai/ui';
```

- [React Spectrum CheckboxGroup ↗](https://react-spectrum.adobe.com/react-spectrum/CheckboxGroup.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default (vertical)

```tsx preview
import { Checkbox, CheckboxGroup } from '@geti-ai/ui';

export default function App() {
  return (
    <CheckboxGroup label="Notification channels">
      <Checkbox value="email">Email</Checkbox>
      <Checkbox value="sms">SMS</Checkbox>
      <Checkbox value="push">Push notification</Checkbox>
    </CheckboxGroup>
  );
}
```

### Pre-selected values

```tsx preview
import { Checkbox, CheckboxGroup } from '@geti-ai/ui';

export default function App() {
  return (
    <CheckboxGroup label="Features" defaultValue={['annotations', 'models']}>
      <Checkbox value="annotations">Annotations</Checkbox>
      <Checkbox value="models">Models</Checkbox>
      <Checkbox value="datasets">Datasets</Checkbox>
    </CheckboxGroup>
  );
}
```

### Horizontal layout

```tsx preview
import { Checkbox, CheckboxGroup } from '@geti-ai/ui';

export default function App() {
  return (
    <CheckboxGroup label="View options" orientation="horizontal">
      <Checkbox value="grid">Grid</Checkbox>
      <Checkbox value="list">List</Checkbox>
      <Checkbox value="map">Map</Checkbox>
    </CheckboxGroup>
  );
}
```

### Disabled group

```tsx preview
import { Checkbox, CheckboxGroup } from '@geti-ai/ui';

export default function App() {
  return (
    <CheckboxGroup label="Locked options" isDisabled>
      <Checkbox value="a">Option A</Checkbox>
      <Checkbox value="b">Option B</Checkbox>
    </CheckboxGroup>
  );
}
```

## Responsibility

`CheckboxGroup` is a thin wrapper around React Spectrum's `CheckboxGroup`. All props pass through unchanged. The wrapper provides a `@geti-ai/ui` import path and Geti theming.

## Variants and states

| Prop | Options | Default |
|---|---|---|
| `orientation` | `'vertical' \| 'horizontal'` | `'vertical'` |
| `isDisabled` | boolean | `false` |
| `isReadOnly` | boolean | `false` |

## Props (deep dive)

### Content

- `children`: `Checkbox` components. Each must have a `value` string.
- `label`: visible group label.
- `description`: optional helper text below the group.
- `errorMessage`: error text shown when `validationState="invalid"`.

### Value

- `value`: controlled array of selected values.
- `defaultValue`: uncontrolled initial selection array.

### Events

- `onChange`: `(values: string[]) => void` — called with full updated selection.

### Validation

- `validationState`: `'valid' | 'invalid'`.
- `isRequired`: marks at least one selection as required.

### Accessibility

- the group renders a `<fieldset>` with a `<legend>` containing the `label`,
- never skip the `label` prop — it is required for accessible grouping.

## Related components

- `Checkbox` — use for a single standalone boolean.
- `RadioGroup` — use when only one selection is allowed.
- `ToggleButtons` — use for mutually exclusive view/mode switching (not a form field).
