# BBox size distribution chart (composition)

`BBoxSizeDistributionChart` helps detect small-object bias and shape skew in detection datasets.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `series` | `ScatterSeriesConfig[]` | — | Per-series data; each has `name` and `data` (array of `{ width, height }` objects) |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `markerSize` | `number` | `8` | Dot size for scatter points |
| `xLogScale` | `boolean` | `false` | Use logarithmic scale on the X axis |
| `yLogScale` | `boolean` | `false` | Use logarithmic scale on the Y axis |
| `xAxisProps` | `XAxisProps` | — | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | — | Extra props forwarded to the Recharts `YAxis` |

## Variant — box width vs height scatter

```tsx preview
import { BBoxSizeDistributionChart } from '@geti-ai/charts';

const points = [
  { width: 12, height: 15 },
  { width: 18, height: 20 },
  { width: 24, height: 27 },
  { width: 36, height: 32 },
  { width: 45, height: 40 },
  { width: 60, height: 55 },
  { width: 74, height: 68 },
  { width: 92, height: 84 },
  { width: 120, height: 110 },
  { width: 154, height: 136 },
  { width: 205, height: 182 },
  { width: 280, height: 236 },
];

export default function App() {
  return (
    <BBoxSizeDistributionChart
      series={[{ name: 'All boxes', data: points }]}
      aria-label="Bounding box width versus height"
      height={380}
    />
  );
}
```

```tsx preview

const train = [
  { width: 14, height: 16 },
  { width: 20, height: 24 },

  { width: 58, height: 53 },
  { width: 78, height: 70 },

  { width: 188, height: 163 },
];

const validation = [
  { width: 12, height: 14 },
  { width: 18, height: 21 },
  { width: 26, height: 30 },
  { width: 38, height: 35 },
  { width: 52, height: 47 },
  { width: 71, height: 63 },

```
