# Icon

`Icon` wraps Spectrum icons so they can be styled and sized consistently in Geti UI components.

```tsx
import { Icon } from '@geti-ui/ui';
import { Filter } from '@geti-ui/ui/icons';
```

- [React Spectrum Icon ↗](https://react-spectrum.adobe.com/react-spectrum/Icon.html)
- [Geti UI Theming](/components/theming)

## Examples

### Basic icon wrapper

```tsx preview
import { Icon } from '@geti-ui/ui';
import { Filter } from '@geti-ui/ui/icons';

export default function App() {
  return (
    <Icon>
      <Filter />
    </Icon>
  );
}
```

### Inline with text

```tsx preview
import { ActionButton, Icon, Text } from '@geti-ui/ui';
import { Filter } from '@geti-ui/ui/icons';

export default function App() {
  return (
    <ActionButton>
      <Icon marginStart="size-100">
        <Filter />
      </Icon>
      <Text>Filter</Text>
    </ActionButton>
  );
}
```

## Responsibility

`Icon` is a thin wrapper around React Spectrum `Icon`. It keeps icon composition on the `@geti-ui/ui` entrypoint and avoids importing directly from `@adobe/react-spectrum`.
