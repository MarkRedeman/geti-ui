# Area chart (composition)

`AreaChart` is a ready-to-use composition with themed defaults, optional stacking, and optional fade gradients.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of data objects for the chart |
| `series` | `AreaSeriesConfig[]` | — | Series definitions; each entry has `dataKey`, `name`, `color?`, `stacked?`, `fade?` |
| `xAxisKey` | `string` | — | Key in each data object for the X axis |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `false` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `xScale` | `ScaleConfig` | — | X axis scale config (e.g. `{ scale: 'time' }`) |
| `yScale` | `ScaleConfig` | — | Y axis scale config (e.g. `{ scale: 'log', domain: [1, 'auto'] }`) |
| `xAxisProps` | `XAxisProps` | — | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | — | Extra props forwarded to the Recharts `YAxis` |

## Variant — `stacked` and `fade` off

```tsx preview
import { AreaChart } from '@geti-ui/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      series={[
        { dataKey: 'train', name: 'Train IoU', stacked: false, fade: false },
        { dataKey: 'val', name: 'Val IoU', stacked: false, fade: false },
      ]}
      aria-label="IoU unstacked no fade"
      height={260}
    />
  );
}
```

## Variant — `stacked` and `fade` on

```tsx preview
import { AreaChart } from '@geti-ui/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      series={[
        { dataKey: 'train', name: 'Train IoU', stacked: true, fade: true },
        { dataKey: 'val', name: 'Val IoU', stacked: true, fade: true },
      ]}
      aria-label="IoU over training steps"
      height={260}
    />
  );
}
```

`series[].fade` uses a vertical gradient fill (strong near the line, fading to transparent at the bottom), matching the custom-chart style.

## Variant — train / validation / test palette

```tsx preview
import { AreaChart, withDatasetSubsetPalette } from '@geti-ui/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41, test: 0.39 },
  { step: 2, train: 0.51, val: 0.46, test: 0.44 },
  { step: 3, train: 0.62, val: 0.57, test: 0.53 },
  { step: 4, train: 0.71, val: 0.65, test: 0.60 },
];

export default function App() {
  const series = withDatasetSubsetPalette([
    { dataKey: 'train', name: 'Training', fade: true },
    { dataKey: 'val', name: 'Validation', fade: true },
    { dataKey: 'test', name: 'Test', fade: true },
  ]);

  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      showLegend
      series={series}
      aria-label="Training/validation/test IoU"
      height={260}
    />
  );
}
```

## Variant — time-based X axis

```tsx preview
import { AreaChart } from '@geti-ui/charts';

const data = [
  { ts: Date.parse('2026-01-01T00:00:00Z'), utilization: 0.42 },
  { ts: Date.parse('2026-01-02T00:00:00Z'), utilization: 0.55 },
  { ts: Date.parse('2026-01-03T00:00:00Z'), utilization: 0.63 },
  { ts: Date.parse('2026-01-04T00:00:00Z'), utilization: 0.58 },
  { ts: Date.parse('2026-01-05T00:00:00Z'), utilization: 0.71 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="ts"
      xScale={{ scale: 'time' }}
      xAxisProps={{
        type: 'number',
        domain: ['dataMin', 'dataMax'],
        tickFormatter: (v) => new Date(Number(v)).toLocaleDateString('en-US', { month: 'short', day: 'numeric' }),
      }}
      series={[{ dataKey: 'utilization', name: 'GPU utilization', fade: true }]}
      aria-label="Utilization over time"
      height={260}
    />
  );
}
```
