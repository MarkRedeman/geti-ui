# Media grid thumbnail item

`MediaGridThumbnailItem` is the default thumbnail-focused item implementation for `MediaGrid`.

It wraps `MediaGridItem` and renders image content with `object-fit: cover`.

Use it when your application needs a fast default item renderer and you only want to customize overlays/actions.

```tsx
import { MediaGridThumbnailItem } from '@geti-ui/blocks';
```

## Basic

```tsx preview
import { ThemeProvider } from '@geti-ui/ui';
import { MediaGridThumbnailItem } from '@geti-ui/blocks';

export default function App() {
  return (
    <ThemeProvider>
      <div style={{ width: 220, height: 220 }}>
        <MediaGridThumbnailItem
          src="https://picsum.photos/seed/thumbnail-item-basic/400/400"
          alt="Thumbnail item"
        />
      </div>
    </ThemeProvider>
  );
}
```

## `MediaGridThumbnailItem` props

`MediaGridThumbnailItem` extends `MediaGridItem` props (except `children`) and adds thumbnail-specific source props.

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `src` | `string` | No | Thumbnail image URL. |
| `alt` | `string` | No | Image alt text. Default: `"Media item thumbnail"`. |
| `isSelected` | `boolean` | No | Selected visual state. |
| `isPlaceholder` | `boolean` | No | Placeholder visual state. |
| `onPress` | `() => void` | No | Press interaction handler. |
| `onDoublePress` | `() => void` | No | Double-press interaction handler. |
| `topLeft` | `ReactNode` | No | Top-left overlay slot. |
| `topRight` | `ReactNode` | No | Top-right overlay slot. |
| `bottomLeft` | `ReactNode` | No | Bottom-left overlay slot. |
| `bottomRight` | `ReactNode` | No | Bottom-right overlay slot. |

## With overlays and selection

```tsx preview
import { useState } from 'react';
import { ThemeProvider } from '@geti-ui/ui';
import {
  MediaGridItemCheckbox,
  MediaGridItemInfo,
  MediaGridItemMenu,
  MediaGridThumbnailItem,
} from '@geti-ui/blocks';

export default function App() {
  const [selected, setSelected] = useState(true);

  return (
    <ThemeProvider>
      <div style={{ width: 240, height: 240 }}>
        <MediaGridThumbnailItem
          src="https://picsum.photos/seed/thumbnail-item-overlays/400/400"
          alt="Thumbnail with overlays"
          isSelected={selected}
          onPress={() => setSelected((v) => !v)}
          topLeft={
            <MediaGridItemCheckbox
              ariaLabel="Select thumbnail"
              isSelected={selected}
              onChange={() => setSelected((v) => !v)}
            />
          }
          topRight={
            <MediaGridItemMenu
              actions={[
                { key: 'open', label: 'Open' },
                { key: 'download', label: 'Download' },
                { key: 'delete', label: 'Delete' },
              ]}
              onAction={(key) => console.log(key)}
            />
          }
          bottomLeft={<MediaGridItemInfo>0.92</MediaGridItemInfo>}
          bottomRight={<MediaGridItemInfo>#1</MediaGridItemInfo>}
        />
      </div>
    </ThemeProvider>
  );
}
```

## Integration notes

- Treat this as the default rendering baseline for dataset/media-browser pages.
- Keep `src` optional in virtualized grids to support placeholder rows while pages are still loading.
- Switch to `MediaGridItem` only when you need full custom media rendering behavior.
