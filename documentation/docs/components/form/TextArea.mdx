# TextArea

TextArea is a multi-line text input for longer freeform content. Use it for descriptions, comments, notes, and any field where the user needs more than a single line.

```tsx
import { TextArea } from '@geti/ui';
```

- [React Spectrum TextArea ↗](https://react-spectrum.adobe.com/react-spectrum/TextArea.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { TextArea } from '@geti/ui';

export default function App() {
  return <TextArea label="Description" />;
}
```

### With placeholder

```tsx preview
import { TextArea } from '@geti/ui';

export default function App() {
  return (
    <TextArea
      label="Project notes"
      placeholder="Add any additional context about this project..."
    />
  );
}
```

### With character limit

```tsx preview
import { TextArea } from '@geti/ui';

export default function App() {
  return (
    <TextArea
      label="Short description"
      maxLength={200}
      description="Max 200 characters"
    />
  );
}
```

### Disabled

```tsx preview
import { TextArea } from '@geti/ui';

export default function App() {
  return <TextArea label="Notes" isDisabled defaultValue="Read-only content" />;
}
```

## Responsibility

`TextArea` is a thin wrapper around React Spectrum's `TextArea`. All props pass through unchanged. The wrapper provides a `@geti/ui` import path and Geti theming.

## Variants and states

| State | Prop | Notes |
|---|---|---|
| Disabled | `isDisabled` | Non-interactive |
| Read-only | `isReadOnly` | Visible but not editable |
| Invalid | `validationState="invalid"` | Shows error styling |
| Required | `isRequired` | Required in form context |

## Props (deep dive)

### Content

- `label`: visible field label.
- `placeholder`: hint text shown when empty.
- `description`: helper text below the field.
- `errorMessage`: error text when `validationState="invalid"`.

### Value

- `value`: controlled string value.
- `defaultValue`: uncontrolled initial value.
- `maxLength`: character limit.

### Events

- `onChange`: `(value: string) => void`.

### Accessibility

- always provide a `label` — text areas must have an accessible name,
- `maxLength` is exposed as `aria-describedby` helper text.

## Related components

- `TextField` — use for single-line text inputs.
- `PasswordField` — use for sensitive single-line text.
