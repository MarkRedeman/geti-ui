# SearchField

SearchField is a text input purpose-built for search and filter interactions. It includes a magnifier icon and a clear button, and fires `onSubmit` on Enter. Use it for any live or submitted search pattern.

```tsx
import { SearchField } from '@geti-ui/ui';
```

- [React Spectrum SearchField ↗](https://react-spectrum.adobe.com/react-spectrum/SearchField.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { SearchField } from '@geti-ui/ui';

export default function App() {
  return <SearchField label="Search datasets" />;
}
```

### With placeholder

```tsx preview
import { SearchField } from '@geti-ui/ui';

export default function App() {
  return (
    <SearchField
      label="Search"
      placeholder="Filter by name or tag..."
    />
  );
}
```

### Disabled

```tsx preview
import { SearchField } from '@geti-ui/ui';

export default function App() {
  return <SearchField label="Search" isDisabled />;
}
```

## Responsibility

`SearchField` is a thin wrapper around React Spectrum's `SearchField` with ref forwarding via `TextFieldRef`. The wrapper provides a `@geti-ui/ui` import path, Geti theming, and allows parent components to programmatically focus or read the field.

## Variants and states

| State | Prop | Notes |
|---|---|---|
| Disabled | `isDisabled` | Input locked |
| Read-only | `isReadOnly` | Visible but not editable |

## Props (deep dive)

### Content

- `label`: accessible label (can be visually hidden with `labelPosition="side"` or `aria-label`).
- `placeholder`: hint text shown when the field is empty.

### Value

- `value`: controlled string value.
- `defaultValue`: uncontrolled initial value.

### Events

- `onChange`: `(value: string) => void` — fires on every keystroke.
- `onSubmit`: `(value: string) => void` — fires when the user presses Enter.
- `onClear`: called when the user presses the built-in clear (✕) button.

### Accessibility

- renders with `role="searchbox"` and a search landmark,
- always provide either a visible `label` or an `aria-label` when the label is hidden.
