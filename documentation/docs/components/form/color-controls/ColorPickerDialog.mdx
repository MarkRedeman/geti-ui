# ColorPickerDialog

`ColorPickerDialog` is a color picker that combines a swatch trigger with a popover containing a `ColorEditor` (color area, hue slider, and color fields) and a `ColorSwatchPicker` for quick preset selection.

Built on top of React Spectrum's [`ColorPicker`](https://react-spectrum.adobe.com/v3/ColorPicker.html) and `ColorEditor`.

```tsx
import { ColorPickerDialog } from '@geti-ui/ui';
```

- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { ColorPickerDialog } from '@geti-ui/ui';

export default function App() {
  return (
    <ColorPickerDialog color="#ff0000" label="Pick Color" onColorChange={(hex) => console.log(hex)} />
  );
}
```

### Custom label

```tsx preview

  return (
    <ColorPickerDialog
      color="#5100FF"
      label="Custom Palette"
      swatches={['#A00', '#f80', '#080', '#08f', '#088', '#008']}

    />
  );
}
```

## Responsibility

`ColorPickerDialog` is a thin wrapper around React Spectrum's `ColorPicker` with a `ColorEditor` and `ColorSwatchPicker`. It:

- Renders a swatch trigger button that opens a popover on click.
- Displays a `ColorEditor` (with `hideAlphaChannel`) for full color editing via a color area, hue slider, and channel fields.
- Shows a `ColorSwatchPicker` below the editor populated with `DISTINCT_COLORS` by default, or a custom set of swatches.
- Accepts a `color` string prop (any CSS color) and calls `onColorChange` with a hex string on every change.
- Falls back to red (`hsb(0, 100%, 100%)`) if an unparseable color string is provided.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `color` | `string` | `undefined` (red fallback) | Any CSS color string. Sets the initial color value. |
| `onColorChange` | `(hex: string) => void` | — | Called with the chosen hex color on every change. |
| `label` | `string` | `"Pick Color"` | Text shown on the trigger button. |
| `swatches` | `string[]` | `DISTINCT_COLORS` | Array of hex color strings for preset swatches. |

## Related components

- [`ColorEditor`](https://react-spectrum.adobe.com/v3/ColorPicker.html#coloreditor): the built-in Spectrum color editing panel (color area + hue slider + fields).
- [`ColorSwatchPicker`](/components/form/color-controls/ColorSwatchPicker): preset palette below the editor.
- [`ColorSwatch`](/components/form/color-controls/ColorSwatch): individual color preview used in the trigger and swatches.
