# Annotation

SVG annotation renderer for overlaying shapes on images and canvases. Supports bounding boxes, circles, polygons, and oriented (rotated) bounding boxes with interactive hover highlighting, click-to-select, and a dimming mask that focuses attention on the hovered annotation.

```tsx
import {
  Annotations,
  AnnotationShape,
  HoveredProvider,
  HoverableAnnotation,
  SelectedProvider,
  SelectableAnnotation,
  MaskAnnotations,
  PredictedAnnotation,
  useAnnotation,
  useHoveredAnnotation,
  useSelectedAnnotation,
} from '@geti-ui/blocks';
```

## Basic usage - annotations on an image

The simplest way to render annotations is the composed `<Annotations>` component. Pass it an array of annotation objects plus the coordinate-space dimensions. Hover and selection are enabled by default.

```tsx preview
import { Annotations } from '@geti-ui/blocks';
import { ThemeProvider, View, Text, Flex } from '@geti-ui/ui';

const IMAGE_WIDTH = 640;
const IMAGE_HEIGHT = 480;

const annotations = [
  {
    id: 'car-1',
    labels: [{ id: 'l1', name: 'Car', color: '#00ff88' }],
    shape: { type: 'bounding-box', x: 60, y: 80, width: 200, height: 140 },
  },
  {
    id: 'wheel-1',
    labels: [{ id: 'l2', name: 'Wheel', color: '#ff4444' }],
    shape: { type: 'circle', cx: 420, cy: 300, r: 60 },
  },
  {
    id: 'road-1',
    labels: [{ id: 'l3', name: 'Road', color: '#4488ff' }],
    shape: {
      type: 'polygon',
      points: [
        { x: 0, y: 400 },
        { x: 640, y: 400 },
        { x: 640, y: 480 },
        { x: 0, y: 480 },
      ],
    },
  },
  {
    id: 'sign-1',
    labels: [{ id: 'l4', name: 'Sign', color: '#ffdd00' }],
    shape: { type: 'oriented-bounding-box', cx: 520, cy: 120, width: 80, height: 40, angle: 15 },
  },
];

export default function App() {
  return (
    <ThemeProvider>
      <View UNSAFE_style={{ background: '#1a1a2e', padding: 16, borderRadius: 8 }}>
        <Text UNSAFE_style={{ color: '#888', fontSize: 12, marginBottom: 8, display: 'block' }}>
          Hover over shapes to highlight them. Click to select (shift-click for multi-select).
        </Text>
        <div style={{ position: 'relative', width: IMAGE_WIDTH, height: IMAGE_HEIGHT }}>
          <img
            src="https://picsum.photos/640/480"
            width={IMAGE_WIDTH}
            height={IMAGE_HEIGHT}
            alt="Sample"
            style={{ display: 'block', position: 'absolute', top: 0, left: 0 }}
          />
          <div style={{ position: 'absolute', top: 0, left: 0 }}>
            <Annotations
              annotations={annotations}
              width={IMAGE_WIDTH}
              height={IMAGE_HEIGHT}
            />
          </div>
        </div>
      </View>
    </ThemeProvider>
  );
}
```

## Non-interactive (read-only) mode

Set `interactive={false}` to disable hover highlighting, selection, and the dimming mask. Useful for thumbnails or read-only previews.

```tsx preview
import { Annotations } from '@geti-ui/blocks';
import { ThemeProvider, View } from '@geti-ui/ui';

const W = 320;
const H = 240;

const annotations = [
  {
    id: 'a',
    labels: [{ id: 'l1', name: 'Object', color: '#00ff88' }],
    shape: { type: 'bounding-box', x: 40, y: 30, width: 120, height: 80 },
  },
  {
    id: 'b',
    labels: [{ id: 'l2', name: 'Region', color: '#ff4444' }],
    shape: { type: 'circle', cx: 240, cy: 160, r: 40 },
  },
];

export default function App() {
  return (
    <ThemeProvider>
      <View UNSAFE_style={{ background: '#1a1a2e', padding: 16, borderRadius: 8 }}>
        <div style={{ position: 'relative', width: W, height: H }}>
          <img
            src="https://picsum.photos/320/240"
            width={W}
            height={H}
            alt="Thumbnail"
            style={{ display: 'block' }}
          />
          <div style={{ position: 'absolute', top: 0, left: 0 }}>
            <Annotations annotations={annotations} width={W} height={H} interactive={false} />
          </div>
        </div>
      </View>
    </ThemeProvider>
  );
}
```

## Composing primitives

For full control, use the individual building blocks instead of the composed `<Annotations>` component. This lets you customize which behaviors each annotation gets (e.g. hover-only without selection, or prediction styling on specific annotations).

```tsx preview
import {
  AnnotationShape,
  HoveredProvider,
  HoverableAnnotation,
  MaskAnnotations,
  AnnotationContext,
} from '@geti-ui/blocks';
import { ThemeProvider, View, Text } from '@geti-ui/ui';

const W = 640;
const H = 400;

const annotations = [
  {
    id: 'a',
    labels: [{ id: 'l1', name: 'Detection A', color: '#00ff88' }],
    shape: { type: 'bounding-box', x: 50, y: 60, width: 180, height: 120 },
  },
  {
    id: 'b',
    labels: [{ id: 'l2', name: 'Detection B', color: '#ff4444' }],
    shape: { type: 'circle', cx: 400, cy: 200, r: 70 },
  },
  {
    id: 'c',
    labels: [{ id: 'l3', name: 'Detection C', color: '#4488ff' }],
    shape: {
      type: 'polygon',
      points: [
        { x: 300, y: 300 },
        { x: 500, y: 280 },
        { x: 520, y: 380 },
        { x: 280, y: 390 },
      ],
    },
  },
];

const SVG_STYLES = {
  fillOpacity: 0.1,
  fill: 'currentColor',
  stroke: 'currentColor',
  strokeLinecap: 'round',
  strokeWidth: 1,
};

export default function App() {
  return (
    <ThemeProvider>
      <View UNSAFE_style={{ background: '#1a1a2e', padding: 16, borderRadius: 8 }}>
        <Text UNSAFE_style={{ color: '#888', fontSize: 12, marginBottom: 8, display: 'block' }}>
          Custom composition: hover-only (no selection). Hover to see the mask dim effect.
        </Text>
        <div style={{ position: 'relative', width: W, height: H }}>
          <img
            src="https://picsum.photos/640/400"
            width={W}
            height={H}
            alt="Sample"
            style={{ display: 'block', position: 'absolute', top: 0, left: 0 }}
          />
          <div style={{ position: 'absolute', top: 0, left: 0 }}>
            <HoveredProvider>
              <svg width={W} height={H} style={SVG_STYLES}>
                <MaskAnnotations annotations={annotations} width={W} height={H}>
                  {annotations.map((annotation) => (
                    <AnnotationContext.Provider key={annotation.id} value={annotation}>
                      <HoverableAnnotation>
                        <AnnotationShape />
                      </HoverableAnnotation>
                    </AnnotationContext.Provider>
                  ))}
                </MaskAnnotations>
              </svg>
            </HoveredProvider>
          </div>
        </div>
      </View>
    </ThemeProvider>
  );
}
```

## Interactions

| Input | Action |
| --- | --- |
| Pointer enter on shape | Highlights the shape and dims the rest of the canvas |
| Pointer leave | Removes highlight, canvas returns to normal |
| Click on shape | Selects the annotation (replaces previous selection) |
| Shift + click | Adds or removes the annotation from the current selection |

## Customization via CSS custom properties

The annotation renderer uses CSS custom properties for all visual styling. Override these on a parent element or on the `<svg>` to customize appearance:

| Property | Default | Description |
| --- | --- | --- |
| `--annotation-fill` | `currentColor` | Shape fill color |
| `--annotation-fill-opacity` | `0.1` | Shape fill opacity |
| `--annotation-stroke` | `currentColor` | Shape stroke color |
| `--annotation-border-opacity` | `1` | Stroke opacity |
| `--annotation-hover-fill` | `yellow` | Fill color when hovered |
| `--annotation-selected-stroke` | `#00c7fd` | Stroke color when selected |
| `--annotation-selected-fill-opacity` | `0.7` | Fill opacity when selected |
| `--annotation-prediction-fill` | `#34d399` | Fill color for prediction annotations |
| `--zoom-scale` | `1` | Current zoom scale (set by `ZoomTransform`). Used in `calc()` for scale-invariant strokes. |

### Interactive color controls

Use `ColorPickerDialog` from `@geti-ui/ui` alongside annotations to let users change fill, stroke, and hover colors in real time. The pickers update CSS custom properties on the annotation container.

```tsx preview
import { useState } from 'react';
import { Annotations } from '@geti-ui/blocks';
import { ThemeProvider, View, Text, Flex, ColorPickerDialog, Slider } from '@geti-ui/ui';

const W = 640;
const H = 400;

const annotations = [
  {
    id: 'bbox-1',
    labels: [{ id: 'l1', name: 'Car', color: '#00ff88' }],
    shape: { type: 'bounding-box', x: 40, y: 50, width: 200, height: 140 },
  },
  {
    id: 'circle-1',
    labels: [{ id: 'l2', name: 'Wheel', color: '#ff4444' }],
    shape: { type: 'circle', cx: 400, cy: 200, r: 70 },
  },
  {
    id: 'polygon-1',
    labels: [{ id: 'l3', name: 'Road', color: '#4488ff' }],
    shape: {
      type: 'polygon',
      points: [
        { x: 200, y: 300 },
        { x: 550, y: 280 },
        { x: 580, y: 380 },
        { x: 180, y: 390 },
      ],
    },
  },
];

export default function App() {
  const [fillColor, setFillColor] = useState('#ffffff');
  const [strokeColor, setStrokeColor] = useState('#ffffff');
  const [hoverFill, setHoverFill] = useState('#ffff00');
  const [fillOpacity, setFillOpacity] = useState(0.1);

  const customProperties = {
    '--annotation-fill': fillColor,
    '--annotation-stroke': strokeColor,
    '--annotation-fill-opacity': fillOpacity,
    '--annotation-hover-fill': hoverFill,
  };

  return (
    <ThemeProvider>
      <View UNSAFE_style={{ background: '#1a1a2e', padding: 16, borderRadius: 8 }}>
        <Flex direction="row" gap="size-300" alignItems="end" wrap UNSAFE_style={{ marginBottom: 12 }}>
          <Flex direction="column" gap="size-50">
            <Text UNSAFE_style={{ color: '#aaa', fontSize: 11 }}>Fill</Text>
            <ColorPickerDialog color={fillColor} onColorChange={setFillColor} label={fillColor} />
          </Flex>
          <Flex direction="column" gap="size-50">
            <Text UNSAFE_style={{ color: '#aaa', fontSize: 11 }}>Stroke</Text>
            <ColorPickerDialog color={strokeColor} onColorChange={setStrokeColor} label={strokeColor} />
          </Flex>
          <Flex direction="column" gap="size-50">
            <Text UNSAFE_style={{ color: '#aaa', fontSize: 11 }}>Hover fill</Text>
            <ColorPickerDialog color={hoverFill} onColorChange={setHoverFill} label={hoverFill} />
          </Flex>
          <Slider
            label="Fill opacity"
            value={fillOpacity}
            onChange={setFillOpacity}
            minValue={0}
            maxValue={1}
            step={0.05}
            width="size-1600"
          />
        </Flex>
        <div style={{ position: 'relative', width: W, height: H }}>
          <img
            src="https://picsum.photos/640/400"
            width={W}
            height={H}
            alt="Sample"
            style={{ display: 'block', position: 'absolute', top: 0, left: 0 }}
          />
          <div style={{ position: 'absolute', top: 0, left: 0, ...customProperties }}>
            <Annotations annotations={annotations} width={W} height={H} />
          </div>
        </div>
      </View>
    </ThemeProvider>
  );
}
```

## Props

### Annotations

The composed component that renders all annotations with full interactivity.

| Prop | Type | Default | Required | Description |
| --- | --- | --- | --- | --- |
| `annotations` | `ReadonlyArray<Annotation>` | - | Yes | The annotations to render. |
| `width` | `number` | - | Yes | Natural width of the coordinate space (e.g. image width in pixels). |
| `height` | `number` | - | Yes | Natural height of the coordinate space (e.g. image height in pixels). |
| `interactive` | `boolean` | `true` | No | When `true`, hover highlighting, click-to-select, and the dimming mask are enabled. |

### AnnotationShape

Renders the SVG primitive for a single annotation's shape.

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `annotation` | `Annotation` | No | The annotation to render. When omitted, reads from the nearest `AnnotationContext`. |

### MaskAnnotations

SVG mask that dims all annotations except the hovered one.

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `annotations` | `ReadonlyArray<Annotation>` | Yes | The full set of annotations (needed to build the mask). |
| `width` | `number` | Yes | Coordinate-space width. |
| `height` | `number` | Yes | Coordinate-space height. |
| `children` | `ReactNode` | Yes | Annotation elements to render inside the mask. |

### PredictedAnnotation

Wrapper that applies prediction styling when labels include predictions.

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `annotation` | `Annotation` | No | The annotation to check. When omitted, reads from context. |
| `children` | `ReactNode` | Yes | Content to wrap. |

## Hooks

| Hook | Returns | Description |
| --- | --- | --- |
| `useAnnotation()` | `Annotation` | Reads the current annotation from context. Throws if used outside an `AnnotationContext.Provider`. |
| `useHoveredAnnotation()` | `string \| null` | Returns the ID of the currently hovered annotation, or `null`. |
| `useSetHoveredAnnotation()` | `Dispatch<SetStateAction<string \| null>>` | Setter to programmatically change the hovered annotation. |
| `useIsHovered()` | `boolean` | Returns `true` when any annotation is hovered. |
| `useSelectedAnnotation()` | `Set<string>` | Returns the set of currently selected annotation IDs. |
| `useSetSelectedAnnotations()` | `Dispatch<SetStateAction<Set<string>>>` | Setter to programmatically change the selected annotations. |

## Types

```ts
type AnnotationPoint = { x: number; y: number };

type BoundingBox = {
  type: 'bounding-box';
  x: number; y: number;
  width: number; height: number;
};

type OrientedBoundingBox = {
  type: 'oriented-bounding-box';
  cx: number; cy: number;
  width: number; height: number;
  angle: number; // degrees
};

type CircleShape = {
  type: 'circle';
  cx: number; cy: number;
  r: number;
};

type PolygonShape = {
  type: 'polygon';
  points: AnnotationPoint[];
};

type Shape = BoundingBox | OrientedBoundingBox | CircleShape | PolygonShape;
type ShapeType = Shape['type'];

type AnnotationLabel = {
  id: string;
  name: string;
  color: string;
  isPrediction?: boolean;
  score?: number;
};

type Annotation = {
  id: string;
  labels: ReadonlyArray<AnnotationLabel>;
  shape: Shape;
};
```
