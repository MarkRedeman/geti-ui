# VirtualizedHorizontalGrid

> [!WARNING]
> Deprecated: migrate to `@geti-ui/blocks` media primitives (`MediaRow`, `MediaGrid`, `MediaColumn`).

`VirtualizedHorizontalGrid` renders a horizontally scrolling virtualized grid.

```tsx
import { VirtualizedHorizontalGrid } from '@geti-ui/ui';
```

- [Geti UI Theming](/components/theming)

## Examples

### Horizontal thumbnail carousel

```tsx preview
import { VirtualizedHorizontalGrid } from '@geti-ui/ui';

export default function App() {
  const items = Array.from({ length: 200 }, (_, i) => ({
    id: `img-${i}`,
    title: `Image ${i}`,
  }));

  return (
    <VirtualizedHorizontalGrid
      items={items}
      height="size-2400"
      layoutOptions={{ size: 160, gap: 8 }}
      idFormatter={(item) => item.id}
      textValueFormatter={(item) => item.title}
      renderItem={(item) => (
        <div
          style={{
            width: 160,
            height: 120,
            background: '#2b2b2b',
            borderRadius: 4,
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            fontSize: '0.75rem',
            color: '#aaa',
          }}
        >
          {item.title}
        </div>
      )}
    />
  );
}
```

## Responsibility

`VirtualizedHorizontalGrid` is a Geti-authored component built on `react-aria-components` `Virtualizer` with a custom `HorizontalLayout`. It handles:

- windowed rendering along the horizontal axis,
- accessible `ListBox` semantics with `orientation="horizontal"`,
- stable keying and text-value extraction via your provided formatters.

You control all item rendering - size, thumbnail content, selection state - via `renderItem`.

## Variants and states

The component does not have named visual variants. All visual variation is achieved through `renderItem` and `layoutOptions`.

## Props (deep dive)

### Content

- `items`: typed array of items to render.
- `renderItem(item)`: returns the React node for each grid cell - you own the size and layout.
- `idFormatter(item)`: extracts a unique string key per item.
- `textValueFormatter(item)`: extracts accessible text for screen readers and type-to-select.

### Layout

- `layoutOptions`: `HorizontalLayoutOptions` - controls `size` (item width in px) and `gap` (spacing between items).
- `height`: Spectrum `DimensionValue` for the scroll container height. Defaults to `'100%'`.
- `listBoxItemStyles`: optional inline styles applied to the `ListBoxItem` wrapper around each rendered item.

### Accessibility

The `ListBox` uses `orientation="horizontal"`, so arrow keys scroll through items. Each item's accessible label comes from `textValueFormatter`.
