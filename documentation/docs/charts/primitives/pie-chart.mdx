# Pie chart (primitive)

Use Recharts `PieChart` + `Pie` with Geti primitives when you want complete low-level control.

## Props

Key Recharts props used on the `Pie` primitive:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | required | Key in each data object for the slice value |
| `nameKey` | `string` | ‚Äî | Key used for the slice name in tooltip/legend |
| `outerRadius` | `number \| string` | `'80%'` | Outer radius of the pie |
| `paddingAngle` | `number` | `0` | Gap in degrees between slices |
| `label` | `boolean \| object \| function` | `false` | Render slice labels |
| `activeIndex` | `number \| number[]` | ‚Äî | Index of the slice(s) to render as active (exploded) |
| `activeShape` | `object \| function` | ‚Äî | Shape config for the active slice (e.g. `{ outerRadius: 108 }`) |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |

## Variant ‚Äî default

```tsx preview
import { Cell, Pie, PieChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ai/charts/primitives';

const data = [
  { name: 'Images', value: 52 },
  { name: 'Videos', value: 18 },
  { name: 'Annotations', value: 22 },
  { name: 'Models', value: 8 },
];

const colors = ['var(--spectrum-global-color-blue-500)', 'var(--spectrum-global-color-green-500)', 'var(--spectrum-global-color-yellow-500)', 'var(--spectrum-global-color-purple-500)'];

export default function App() {
  return (
    <GetiResponsiveContainer height={320}>
      <PieChart>
        <Pie data={data} dataKey="value" nameKey="name" outerRadius="80%" isAnimationActive={false}>
          {data.map((_entry, index) => <Cell key={index} fill={colors[index % colors.length]} />)}
        </Pie>
        <GetiTooltip />
        <GetiLegend />
      </PieChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant ‚Äî labels + padding angle

```tsx preview
import { Cell, Pie, PieChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ai/charts/primitives';

const data = [
  { name: 'Train', value: 70 },
  { name: 'Validation', value: 20 },
  { name: 'Test', value: 10 },
];

const colors = ['#22c55e', '#ef4444', '#f59e0b'];

export default function App() {
  return (
    <GetiResponsiveContainer height={320}>
      <PieChart>
        <Pie data={data} dataKey="value" nameKey="name" outerRadius="78%" paddingAngle={2} label isAnimationActive={false}>
          {data.map((_entry, index) => <Cell key={index} fill={colors[index % colors.length]} />)}
        </Pie>
        <GetiTooltip />
        <GetiLegend />
      </PieChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant ‚Äî exploded slice

```tsx preview
import { Cell, Pie, PieChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ai/charts/primitives';

const data = [
  { name: 'Car', value: 4200 },
  { name: 'Person', value: 5600 },
  { name: 'Bicycle', value: 2100 },
  { name: 'Bus', value: 1400 },
];

const colors = ['#60a5fa', '#34d399', '#fbbf24', '#f472b6'];

export default function App() {
  return (
    <GetiResponsiveContainer height={320}>
      <PieChart>
        <Pie
          data={data}
          dataKey="value"
          nameKey="name"
          outerRadius="78%"
          activeIndex={1}
          activeShape={{ outerRadius: 108 }}
          isAnimationActive={false}
        >
          {data.map((_entry, index) => <Cell key={index} fill={colors[index % colors.length]} />)}
        </Pie>
        <GetiTooltip />
        <GetiLegend />
      </PieChart>
    </GetiResponsiveContainer>
  );
}
```
