# Reliability diagram (composition)

`ReliabilityDiagram` compares predicted confidence against observed accuracy to assess calibration.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `{ confidence: number; accuracy: number; samples?: number }[]` | - | Calibration bin data with mean confidence and observed accuracy |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `showPerfectCalibration` | `boolean` | `true` | Render the diagonal perfect-calibration reference line |
| `xAxisProps` | `XAxisProps` | - | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | - | Extra props forwarded to the Recharts `YAxis` |

## Variant - basic calibration curve

```tsx preview
import { ReliabilityDiagram } from '@geti-ui/charts';

const data = [
  { confidence: 0.05, accuracy: 0.12, samples: 54 },
  { confidence: 0.15, accuracy: 0.21, samples: 81 },
  { confidence: 0.25, accuracy: 0.31, samples: 92 },
  { confidence: 0.35, accuracy: 0.40, samples: 120 },
  { confidence: 0.45, accuracy: 0.49, samples: 147 },
  { confidence: 0.55, accuracy: 0.58, samples: 164 },
  { confidence: 0.65, accuracy: 0.66, samples: 171 },
  { confidence: 0.75, accuracy: 0.74, samples: 160 },
  { confidence: 0.85, accuracy: 0.79, samples: 124 },
  { confidence: 0.95, accuracy: 0.84, samples: 88 },
];

export default function App() {
  return (
    <ReliabilityDiagram
      data={data}
      aria-label="Reliability diagram for model calibration"
      height={340}
    />
  );
}
```
