# Primary Tool Thumbnails

Webp preview images for each Geti annotation primary tool.

```tsx
import { BrushTool, PolygonTool, DetectionTool } from '@geti-ui/ui/assets/primary-tools';
```

## Usage

These exports are URL strings (webp image paths). Use them in `<img>` tags or CSS:

```tsx
import { BrushTool, PolygonTool, QuickSelectionTool } from '@geti-ui/ui/assets/primary-tools';

<img src={BrushTool} alt="Brush tool" width={64} height={64} />
<img src={PolygonTool} alt="Polygon tool" width={64} height={64} />
```

## Import path

```ts
import { BrushTool, PolygonTool } from '@geti-ui/ui/assets/primary-tools';

// Or via the combined namespace
import { PrimaryTools } from '@geti-ui/ui/assets';
const { BrushTool } = PrimaryTools;
```
