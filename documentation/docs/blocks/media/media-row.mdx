# Media row

`MediaRow` is a single-row, horizontally virtualized media layout.

Use it for carousel-like timelines and strips (frame pickers, filmstrips, horizontal result browsers) while reusing the same item composition API as `MediaGrid`.

```tsx
import {
  MediaRow,
  MediaGridThumbnailItem,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';
```

## Example with previous/next buttons

```tsx preview
import { useState } from 'react';
import { Button, Flex, ThemeProvider } from '@geti-ai/ui';
import {
  MediaRow,
  MediaGridThumbnailItem,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';

type Media = {
  id: string;
  name: string;
  thumbnailUrl: string;
};

const itemSize = 180;
const gap = 8;

const media: Media[] = Array.from({ length: 150 }, (_, index) => ({
  id: `media-${index + 1}`,
  name: `Image ${index + 1}`,
  thumbnailUrl: `https://picsum.photos/seed/media-row-${index + 1}/400/400`,
}));

export default function App() {
  const [selectedIndex, setSelectedIndex] = useState(0);

  const selectedItem = media[selectedIndex];
  const selectedKeys: MediaGridSelection = selectedItem
    ? new Set([selectedItem.id])
    : new Set();

  const selectRelative = (direction: 'prev' | 'next') => {
    setSelectedIndex((current) => {
      if (direction === 'next') {
        return (current + 1) % media.length;
      }
      return (current - 1 + media.length) % media.length;
    });
  };

  const handleSelectionChange = (keys: MediaGridSelection) => {
    if (keys === 'all') return;

    const [first] = Array.from(keys);
    if (!first) return;

    const index = media.findIndex((item) => item.id === first);
    if (index >= 0) {
      setSelectedIndex(index);
    }
  };

  const renderItem = (ctx: MediaGridRenderContext<Media>) => (
    <MediaGridThumbnailItem
      src={ctx.item?.thumbnailUrl}
      alt={ctx.item?.name ?? 'Loading media'}
      isPlaceholder={ctx.isPlaceholder}
      isSelected={ctx.isSelected}
      onPress={ctx.onPress}
      onDoublePress={ctx.onDoublePress}
    />
  );

  return (
    <ThemeProvider>
      <div style={{ height: 280 }}>
        <Flex alignItems="center" gap="size-100" height="100%" direction="column" width="100%">
          <div style={{ fontSize: 12 }}>Current selected: {selectedItem?.name ?? 'None'}</div>
          <Flex alignItems="center" gap="size-100" width="100%" style={{ minHeight: 0, flex: 1 }}>
            <Button variant="secondary" onPress={() => selectRelative('prev')}>
              Previous
            </Button>

            <div style={{ flex: 1, minWidth: 0, minHeight: 0, height: itemSize }}>
              <MediaRow
                totalItems={media.length}
                getItemAt={(index) => media[index]}
                itemSize={itemSize}
                gap={gap}
                selectionMode="single"
                selectedKeys={selectedKeys}
                onSelectionChange={handleSelectionChange}
                focusOnSelect
                renderItem={renderItem}
              />
            </div>
     
            <Button variant="primary" onPress={() => selectRelative('next')}>
              Next
            </Button>
          </Flex>
        </Flex>
      </div>
    </ThemeProvider>
  );
}
```

## `MediaRow` props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `totalItems` | `number` | Yes | Total number of represented items. |
| `getItemAt` | `(index: number) => T \| undefined` | Yes | Returns item data for index or `undefined` for placeholders. |
| `renderItem` | `(context: MediaGridRenderContext<T>) => ReactNode` | Yes | Renders each virtualized cell. |
| `itemSize` | `number` | No | Square item size in px. Default: `200`. |
| `gap` | `number` | No | Horizontal spacing between items. Default: `8`. |
| `focusOnSelect` | `boolean` | No | When `true`, scrolls the selected item into view after selection changes. Default: `false`. |
| `selectionMode` | `'none' \| 'single' \| 'multiple'` | No | Selection behavior. Default: `'multiple'`. |
| `selectedKeys` | `Set<string> \| 'all'` | No | Controlled selection state. |
| `defaultSelectedKeys` | `Iterable<string>` | No | Initial uncontrolled selection. |
| `onSelectionChange` | `(keys: Set<string> \| 'all') => void` | No | Called when selection changes. |
| `isLoading` | `boolean` | No | Initial/loading state for the row. |
| `emptyState` | `ReactNode` | No | Custom empty-state UI when there are no items. |
| `ariaLabel` | `string` | No | Accessible label for the horizontal row. Default: `'Media row'`. |
| `onItemPress` | `(context: MediaGridRenderContext<T>) => void` | No | Called when an item is pressed. |
| `onItemDoublePress` | `(context: MediaGridRenderContext<T>) => void` | No | Called when an item is double-pressed. |
| `className` | `string` | No | Optional root class. |
| `style` | `CSSProperties` | No | Inline styles for root. |

## Integration notes

- Ensure the row container has a fixed/known height so horizontal virtualization can compute item geometry.
- Keep `itemSize` aligned with your thumbnail design token scale (e.g. 120/180/240).
- Use `focusOnSelect` when row selection changes from external buttons/hotkeys.
- Keep domain actions in `renderItem` overlays; keep row virtualization and selection logic in the row itself.
