# Slider

Slider is a single-handle range control for selecting a numeric value within a bounded range. Use it for settings with continuous or stepped values where drag interaction is more natural than typed input.

```tsx
import { Slider } from '@geti/ui';
```

- [React Spectrum Slider ↗](https://react-spectrum.adobe.com/react-spectrum/Slider.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { Slider } from '@geti/ui';

export default function App() {
  return <Slider label="Opacity" defaultValue={50} minValue={0} maxValue={100} />;
}
```

### Filled track (Geti accent color)

```tsx preview
import { Slider } from '@geti/ui';

export default function App() {
  return <Slider label="Opacity" defaultValue={50} minValue={0} maxValue={100} isFilled />;
}
```

### With step

```tsx preview
import { Slider } from '@geti/ui';

export default function App() {
  return (
    <Slider label="Zoom level" defaultValue={100} minValue={25} maxValue={400} step={25} />
  );
}
```

### Disabled

```tsx preview
import { Slider } from '@geti/ui';

export default function App() {
  return (
    <Slider label="Locked value" defaultValue={50} minValue={0} maxValue={100} isDisabled />
  );
}
```

## Responsibility

`Slider` wraps React Spectrum's `Slider` with a Geti-specific `isFilled` enhancement. When `isFilled` is true, the Geti wrapper applies energy-blue design tokens to the fill track and handle via CSS, overriding Spectrum's default accent color.

## Variants and states

| Prop | Default | Notes |
|---|---|---|
| `isFilled` | `false` | Activates Geti's blue fill track style |
| `isDisabled` | `false` | Freezes the handle |

## Props (deep dive)

### Content

- `label`: visible label above the slider.
- `getValueLabel`: custom renderer for the current value text.

### Value

- `value`: controlled numeric value.
- `defaultValue`: uncontrolled initial value.
- `minValue` / `maxValue`: range bounds.
- `step`: step size for each movement increment.

### Events

- `onChange`: `(value: number) => void` — fires during drag.
- `onChangeEnd`: fires when the user releases the handle.

### Visual

- `isFilled`: applies Geti's blue fill track via CSS class.
- `orientation`: `'horizontal'` (default) or `'vertical'`.

### Accessibility

- always provide a `label` — the slider communicates its value via `aria-valuenow`,
- arrow keys move the handle by `step`; `Page Up`/`Down` by larger increments.

## Related components

- `RangeSlider` — use for dual-handle range selection.
- `NumberField` — use when typed precision input is preferred.
