## Responsibility

The `Checkbox` component is a lightweight wrapper around Adobe's React Spectrum `Checkbox`. It provides a consistent, accessible way to capture boolean input within the Geti UI design system.

### Key Features
- **Binary Toggle**: Perfect for single "On/Off" or "Accept" settings.
- **Indeterminate State**: Supports a visual "dash" state for parent checkboxes in nested selection trees.
- **Form Integration**: Works seamlessly within Spectrum `Form` containers with built-in validation support.
- **Accessibility**: Automatically handles labeling, focus management, and keyboard interaction (Space key).

---

## Variants

### States
Checkboxes support several functional states:
- **Selected**: The checked state, indicating a "true" or "active" value.
- **Indeterminate**: A visual-only state (typically a dash) used when a group of sub-items has a partial selection.
- **Disabled**: Prevents all interaction while remaining visible.
- **Invalid**: Highlights the checkbox to indicate a validation error.

## Props

### Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | The label text for the checkbox. |
| `name` | `string` | — | The name of the input element, used during form submission. |
| `value` | `string` | — | The value of the input element. |

### Value / State

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isSelected` | `boolean` | — | Controlled selection state. |
| `defaultSelected` | `boolean` | — | Initial selection state (uncontrolled). |
| `isIndeterminate` | `boolean` | `false` | Renders a dash instead of a checkmark. |
| `isDisabled` | `boolean` | `false` | Whether the checkbox is disabled. |
| `isReadOnly` | `boolean` | `false` | Whether the checkbox is read-only. |
| `isRequired` | `boolean` | `false` | Whether the checkbox is required for form submission. |

### Events

| Prop | Type | Description |
|------|------|-------------|
| `onChange` | `(isSelected: boolean) => void` | Handler called when the selection state changes. |

### Styles

The `Checkbox` component accepts all standard [Spectrum layout and spacing props](https://react-spectrum.adobe.com/react-spectrum/styling.html).

## Accessibility

- **Labeling**: Every checkbox must have a visible label (via `children`). If a visible label is not possible, an `aria-label` must be provided.
- **Keyboard**: Use `Space` to toggle the checkbox and `Tab` to move focus.
- **Indeterminate State**: Note that `isIndeterminate` is **purely visual**. It does not affect the underlying boolean value of the checkbox.
