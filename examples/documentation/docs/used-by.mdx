# Used By

These Intel products and projects use the Geti UI design system.

## Geti

The flagship visual AI platform for data-centric computer vision. Geti UI was originally extracted from this project's component library.

## Anomalib Studio

Visual interface for anomaly detection model training and deployment powered by [Anomalib](https://github.com/openvinotoolkit/anomalib).

## Geti Instant Learn

Rapid model training interface for quick few-shot learning workflows.

## Geti Tune

Fine-tuning interface for adjusting pre-trained models to specific use cases.

## Physical AI Studio

Development environment for physical AI applications combining vision and robotics.
