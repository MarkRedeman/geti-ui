# Tooltip

Tooltip shows a short text label when the user hovers or focuses a trigger element. Use it to clarify the purpose of icon buttons, truncated labels, or any UI element whose meaning is not immediately obvious.

```tsx
import { Tooltip, TooltipTrigger } from '@geti/ui';
```

## Responsibility

`Tooltip` and `TooltipTrigger` are thin wrappers around React Spectrum's equivalents. `TooltipTrigger` manages show/hide on hover and focus. `Tooltip` renders the floating label. Spectrum handles delays, focus trap avoidance, and ARIA linkage automatically.

## Variants and states

| Prop on `TooltipTrigger` | Notes |
|---|---|
| `placement` | `top` / `bottom` / `start` / `end` and combinations; defaults to `top` |
| `delay` | Show delay in ms (default 1500ms) |
| `isDisabled` | Suppresses the tooltip entirely |

## Props (deep dive)

### TooltipTrigger

- `children`: must be exactly two elements — the trigger and a `Tooltip`.
- `placement`: floating position relative to the trigger.
- `delay`: milliseconds before the tooltip appears (default 1500).
- `closeDelay`: milliseconds before the tooltip hides.
- `isDisabled`: prevents the tooltip from showing.

### Tooltip

- `children`: the tooltip label text or content.

### Accessibility

- `TooltipTrigger` adds `aria-describedby` to the trigger pointing at the `Tooltip` content.
- Only use tooltips for supplementary information — never for information required to complete an action.
- Do not rely solely on tooltips for icon buttons that have no other label; provide `aria-label` on the button too.

## Related components

- `ContextualHelp` — use for longer help text in a popover with a heading, triggered by a dedicated help icon.
- `Popover` — use when the overlay content is richer than a single-line label.

## Example use cases

<details>
  <summary>Typical use cases</summary>

- Icon-only `ActionButton` in a toolbar (`aria-label` + tooltip for sighted and screen-reader users).
- Explaining a disabled button: `isDisabled` + tooltip describing why.
- Column header with truncated text: tooltip shows the full label.
