# IntelBrandedLoading

`IntelBrandedLoading` renders the Intel branded loading animation with a dedicated container. Use it for branded loading moments such as initial app bootstrap or major workflow transitions.

```tsx
import { IntelBrandedLoading } from '@geti-ui/ui';
```

- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { IntelBrandedLoading } from '@geti-ui/ui';

export default function App() {
  return <IntelBrandedLoading />;
}
```

### Fixed height container

```tsx preview
import { IntelBrandedLoading } from '@geti-ui/ui';

export default function App() {
  return (
    <div style={{ height: 220 }}>
      <IntelBrandedLoading />
    </div>
  );
}
```

## Responsibility

`IntelBrandedLoading` is a dedicated Geti component for branded loading visuals.
It wraps the Intel animation in a centered container and handles image sizing.

## Variants and states

This component currently has no variants or states. Use `Loading` when you need
mode, size, or non-branded spinner options.

## Props (deep dive)

`IntelBrandedLoading` currently has no public props.

## Related components

- [`Loading`](/components/feedback/Loading) — use for inline/overlay/fullscreen loading with spinner or intel variant.
- [`ProgressCircle`](/components/feedback/ProgressCircle) — use when only an indeterminate circular indicator is needed.
