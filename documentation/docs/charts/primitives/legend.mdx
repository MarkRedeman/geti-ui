# Legend

`GetiLegend` is the themed legend primitive exported from `@geti-ui/charts/primitives`.

It wraps Recharts `Legend` with Geti typography and default positioning so legend items match chart styling out of the box.

## Props

`GetiLegend` props (exposed by the wrapper):

| Prop | Type | Default | Description |
|---|---|---|---|
| `verticalAlign` | `'top' \| 'middle' \| 'bottom'` | `'bottom'` | Legend position relative to the chart |
| `align` | `'left' \| 'center' \| 'right'` | `'center'` | Horizontal alignment of legend items |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction items are arranged |

All standard Recharts `Legend` props are also accepted and forwarded.

## Import

```ts
import { GetiLegend } from '@geti-ui/charts/primitives';
```

## Usage

```tsx preview
import { Line, LineChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ui/charts/primitives';

const data = [
  { epoch: 1, train: 1.42, val: 1.50 },
  { epoch: 2, train: 1.10, val: 1.20 },
  { epoch: 3, train: 0.87, val: 0.98 },
  { epoch: 4, train: 0.71, val: 0.82 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <LineChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="epoch" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend />
        <Line type="monotone" dataKey="train" name="Train loss" stroke="var(--spectrum-global-color-blue-500)" dot={false} isAnimationActive={false} />
        <Line type="monotone" dataKey="val" name="Val loss" stroke="var(--spectrum-global-color-green-500)" dot={false} isAnimationActive={false} />
      </LineChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant - custom alignment and layout

```tsx preview
import { Line, LineChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ui/charts/primitives';

const data = [
  { epoch: 1, train: 1.42, val: 1.50 },
  { epoch: 2, train: 1.10, val: 1.20 },
  { epoch: 3, train: 0.87, val: 0.98 },
  { epoch: 4, train: 0.71, val: 0.82 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={280}>
      <LineChart data={data} margin={{ top: 8, right: 16, bottom: 32, left: 0 }}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="epoch" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend align="left" verticalAlign="bottom" layout="horizontal" />
        <Line type="monotone" dataKey="train" name="Train loss" stroke="var(--spectrum-global-color-blue-500)" dot={false} isAnimationActive={false} />
        <Line type="monotone" dataKey="val" name="Val loss" stroke="var(--spectrum-global-color-green-500)" dot={false} isAnimationActive={false} />
      </LineChart>
    </GetiResponsiveContainer>
  );
}
```

## Notes

- `GetiLegend` accepts standard Recharts `Legend` props.
- You can override alignment/layout using props like `align`, `verticalAlign`, and `layout`.
- Use together with `GetiTooltip`, `GetiXAxis`, and `GetiYAxis` for a fully themed primitive chart.

## Series highlighting

The legend is the primary control for highlight interactions - hover and click events on legend items drive transient and pinned highlighting. See [Highlights](/charts/highlights).
