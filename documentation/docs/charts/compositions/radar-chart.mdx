# Radar chart (composition)

`RadarChart` is a ready-to-use composition for multi-metric profile comparisons across categories.

Use `radiusAxisProps={{ domain: [0, 1] }}` for normalized scores, and `radiusAxisProps={{ domain: [0, 100] }}` for percentage-style metrics.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of objects, one per category axis |
| `series` | `RadarSeriesConfig[]` | — | Series definitions; each entry has `dataKey`, `name`, `color?`, `fillOpacity?` |
| `categoryKey` | `string` | `'subject'` | Key in each data object for the axis label |
| `filled` | `boolean` | `true` | Fill the area under each radar series |
| `showGrid` | `boolean` | `true` | Show polar grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `angleAxisProps` | `PolarAngleAxisProps` | — | Extra props forwarded to the Recharts `PolarAngleAxis` |
| `radiusAxisProps` | `PolarRadiusAxisProps` | — | Extra props forwarded to the Recharts `PolarRadiusAxis` (e.g. `domain`) |
| `height` | `number` | `300` | Chart height in pixels |

## Variant — default (filled)

```tsx preview
import { RadarChart } from '@geti-ui/charts';

const data = [
  { metric: 'mAP', train: 0.88, val: 0.82 },
  { metric: 'Recall', train: 0.91, val: 0.86 },
  { metric: 'Precision', train: 0.89, val: 0.84 },
  { metric: 'Stability', train: 0.76, val: 0.71 },
  { metric: 'Latency score', train: 0.66, val: 0.69 },
];

export default function App() {
  return (
    <RadarChart
      data={data}
      categoryKey="metric"
      radiusAxisProps={{ domain: [0, 1] }}
      series={[
        { dataKey: 'train', name: 'Train' },
        { dataKey: 'val', name: 'Validation' },
      ]}
      aria-label="Train vs validation radar profile"
      height={360}
    />
  );
}
```

## Variant — train / validation / test palette

```tsx preview
import { RadarChart, withDatasetSubsetPalette } from '@geti-ui/charts';

const data = [
  { metric: 'mAP', train: 0.88, val: 0.82, test: 0.80 },
  { metric: 'Recall', train: 0.91, val: 0.86, test: 0.84 },
  { metric: 'Precision', train: 0.89, val: 0.84, test: 0.83 },
  { metric: 'Stability', train: 0.76, val: 0.71, test: 0.70 },
  { metric: 'Latency score', train: 0.66, val: 0.69, test: 0.68 },
];

export default function App() {
  const series = withDatasetSubsetPalette([
    { dataKey: 'train', name: 'Training', fillOpacity: 0.2 },
    { dataKey: 'val', name: 'Validation', fillOpacity: 0.2 },
    { dataKey: 'test', name: 'Test', fillOpacity: 0.2 },
  ]);

  return (
    <RadarChart
      data={data}
      categoryKey="metric"
      radiusAxisProps={{ domain: [0, 1] }}
      series={series}
      aria-label="Dataset subset radar comparison"
      height={360}
    />
  );
}
```
