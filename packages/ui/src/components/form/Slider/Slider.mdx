# Slider

A numeric input that allows users to select a single value within a fixed range.

```tsx
import { Slider } from '@geti/ui';
```

- [React Spectrum V3 Slider](https://react-spectrum.adobe.com/react-spectrum/Slider.html)

<Stories />

## Responsibility

`Slider` wraps Adobe React Spectrum's `Slider` with a single Geti customisation: when `isFilled={true}`, the `data-filled` CSS attribute triggers a CSS Module override that replaces Spectrum's default fill-track and handle colours with Geti's `--energy-blue` tokens.

`UNSAFE_className` is merged with the Geti module class via `clsx`. All other props pass through to Spectrum unchanged.

### When to use

- To select a single value from a fixed range where the relative position matters as much as the exact number.
- When small incremental adjustments are the dominant interaction pattern.

### When to avoid

- Use [`RangeSlider`](/docs/components-form-rangeslider--docs) when both a minimum and a maximum value must be selected.
- Use [`NumberField`](/docs/components-form-numberfield--docs) when the user needs to type a precise value or the range is large.
- Use [`ProgressBar`](/docs/components-status-progressbar--docs) for read-only status indication.

## States

- **Disabled**: Non-interactive, track and handle dimmed. Use `isDisabled`.
- **Filled**: `isFilled={true}` applies the energy-blue token override to the fill track and handle.
- **Orientation**: `orientation="vertical"` for vertical layouts; horizontal is the default.
- **Formatting**: `formatOptions` accepts `Intl.NumberFormatOptions` for percentages, currency, or units.

## Props

<Controls of={DocStories.Default} />

### Value

| Prop           | Type     | Default | Description                              |
| -------------- | -------- | ------- | ---------------------------------------- |
| `value`        | `number` | —       | The current value (controlled).          |
| `defaultValue` | `number` | —       | The initial value (uncontrolled).        |
| `minValue`     | `number` | `0`     | The minimum allowed value.               |
| `maxValue`     | `number` | `100`   | The maximum allowed value.               |
| `step`         | `number` | `1`     | The increment between selectable values. |

### Content

| Prop             | Type                        | Description                                            |
| ---------------- | --------------------------- | ------------------------------------------------------ |
| `label`          | `ReactNode`                 | The label for the slider.                              |
| `showValueLabel` | `boolean`                   | Whether to display the current value as a label.       |
| `getValueLabel`  | `(value: number) => string` | A function to generate a custom value label.           |
| `formatOptions`  | `Intl.NumberFormatOptions`  | Locale-aware display formatting for the value label.   |
| `contextualHelp` | `ReactNode`                 | A `<ContextualHelp>` element placed next to the label. |

### Events

| Prop          | Type                      | Description                                           |
| ------------- | ------------------------- | ----------------------------------------------------- |
| `onChange`    | `(value: number) => void` | Called continuously while the user drags the slider.  |
| `onChangeEnd` | `(value: number) => void` | Called only when the user releases the slider handle. |

### Styles

| Prop            | Type                         | Description                                                     |
| --------------- | ---------------------------- | --------------------------------------------------------------- |
| `isFilled`      | `boolean`                    | Whether to show a filled track using Geti's energy-blue tokens. |
| `fillOffset`    | `number`                     | The value from which the track fill starts.                     |
| `orientation`   | `'horizontal' \| 'vertical'` | Layout orientation.                                             |
| `width`         | `DimensionValue`             | The width of the slider component.                              |
| `labelPosition` | `'top' \| 'side'`            | The position of the label relative to the slider.               |

## Related components

- `RangeSlider` — use instead when users need to select both a minimum and maximum value on the same track.
- `NumberField` — use instead when users need to type a precise numeric value.
- `ColorSlider` — use instead when adjusting a single color channel (hue, alpha, saturation).
