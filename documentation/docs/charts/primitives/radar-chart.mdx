# Radar chart (primitive)

Use Recharts `RadarChart` primitives when you need full control over polar axes, grid, and radar styling.

Use `PolarRadiusAxis domain={[0, 1]}` for normalized scores, and `PolarRadiusAxis domain={[0, 100]}` for percentage-style metrics.

## Props

Key Recharts props used on the `Radar` and polar axis primitives:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | required | Key in each data object to plot on the `Radar` |
| `name` | `string` | - | Series label shown in tooltip and legend |
| `stroke` | `string` | - | Outline color of the radar polygon |
| `fill` | `string` | - | Fill color of the radar polygon; use `'none'` for outline-only |
| `fillOpacity` | `number` | `0.5` | Opacity of the fill (0–1) |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |
| `domain` | `[min, max]` | - | Value range for `PolarRadiusAxis` (e.g. `[0, 1]` or `[0, 100]`) |

## Variant - default (filled)

```tsx preview
import { PolarAngleAxis, PolarGrid, PolarRadiusAxis, Radar, RadarChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ui/charts/primitives';

const data = [
  { metric: 'mAP', train: 0.88, val: 0.82 },
  { metric: 'Recall', train: 0.91, val: 0.86 },
  { metric: 'Precision', train: 0.89, val: 0.84 },
  { metric: 'Stability', train: 0.76, val: 0.71 },
  { metric: 'Latency score', train: 0.66, val: 0.69 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={360}>
      <RadarChart data={data}>
        <PolarGrid stroke="var(--spectrum-global-color-gray-700)" />
        <PolarAngleAxis dataKey="metric" />
        <PolarRadiusAxis domain={[0, 1]} />
        <Radar name="Train" dataKey="train" stroke="var(--spectrum-global-color-green-500)" fill="var(--spectrum-global-color-green-500)" fillOpacity={0.22} isAnimationActive={false} />
        <Radar name="Validation" dataKey="val" stroke="var(--spectrum-global-color-red-500)" fill="var(--spectrum-global-color-red-500)" fillOpacity={0.22} isAnimationActive={false} />
        <GetiTooltip />
        <GetiLegend />
      </RadarChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant - train / validation / test palette

```tsx preview
import { PolarAngleAxis, PolarGrid, PolarRadiusAxis, Radar, RadarChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ui/charts/primitives';
import { getDatasetSubsetColor } from '@geti-ui/charts';

const data = [
  { metric: 'mAP', train: 0.88, val: 0.82, test: 0.80 },
  { metric: 'Recall', train: 0.91, val: 0.86, test: 0.84 },
  { metric: 'Precision', train: 0.89, val: 0.84, test: 0.83 },
  { metric: 'Stability', train: 0.76, val: 0.71, test: 0.70 },
  { metric: 'Latency score', train: 0.66, val: 0.69, test: 0.68 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={360}>
      <RadarChart data={data}>
        <PolarGrid stroke="var(--spectrum-global-color-gray-700)" />
        <PolarAngleAxis dataKey="metric" />
        <PolarRadiusAxis domain={[0, 1]} />
        <Radar name="Train" dataKey="train" stroke={getDatasetSubsetColor('train')} fill={getDatasetSubsetColor('train')} fillOpacity={0.2} isAnimationActive={false} />
        <Radar name="Validation" dataKey="val" stroke={getDatasetSubsetColor('val')} fill={getDatasetSubsetColor('val')} fillOpacity={0.2} isAnimationActive={false} />
        <Radar name="Test" dataKey="test" stroke={getDatasetSubsetColor('test')} fill={getDatasetSubsetColor('test')} fillOpacity={0.2} isAnimationActive={false} />
        <GetiTooltip />
        <GetiLegend />
      </RadarChart>
    </GetiResponsiveContainer>
  );
}
```
