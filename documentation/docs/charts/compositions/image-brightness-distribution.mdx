# Image brightness distribution (composition)

`ImageBrightnessDistribution` visualizes dataset brightness characteristics and highlights under/over exposure ranges.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `{ binStart: number; binEnd: number; count: number }[]` | ‚Äî | Array of histogram bins with start, end, and count |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `animate` | `boolean` | `false` | Enable entry animation |
| `lowThreshold` | `number` | ‚Äî | Draw a reference line marking under-exposure below this value |
| `highThreshold` | `number` | ‚Äî | Draw a reference line marking over-exposure above this value |
| `xAxisProps` | `XAxisProps` | ‚Äî | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | ‚Äî | Extra props forwarded to the Recharts `YAxis` |

## Variant ‚Äî baseline brightness histogram

```tsx preview
import { ImageBrightnessDistribution } from '@geti-ui/charts';

const data = [
  { binStart: 0, binEnd: 16, count: 28 },
  { binStart: 16, binEnd: 32, count: 46 },
  { binStart: 32, binEnd: 48, count: 67 },
  { binStart: 48, binEnd: 64, count: 91 },
  { binStart: 64, binEnd: 80, count: 122 },
  { binStart: 80, binEnd: 96, count: 144 },
  { binStart: 96, binEnd: 112, count: 162 },
  { binStart: 112, binEnd: 128, count: 183 },
  { binStart: 128, binEnd: 144, count: 176 },
  { binStart: 144, binEnd: 160, count: 148 },
  { binStart: 160, binEnd: 176, count: 117 },
  { binStart: 176, binEnd: 192, count: 86 },
  { binStart: 192, binEnd: 208, count: 58 },
  { binStart: 208, binEnd: 224, count: 33 },
  { binStart: 224, binEnd: 240, count: 16 },
  { binStart: 240, binEnd: 255, count: 7 },
];

export default function App() {
  return (
    <ImageBrightnessDistribution
      data={data}
      aria-label="Image brightness distribution"
      height={320}
    />
  );
}
```
