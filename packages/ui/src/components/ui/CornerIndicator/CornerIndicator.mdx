## Responsibility

`CornerIndicator` is a Geti-specific layout wrapper that overlays a small blue dot in the top-right corner of its children when `isActive={true}`. It is a **decorator**ŌĆöit wraps arbitrary content and conditionally renders the dot without modifying the child.

### How it differs from Spectrum
There is no direct equivalent to a positioned notification dot in Adobe React Spectrum. While Spectrum provides `Badge` and `StatusLight`, they differ from `CornerIndicator` in key ways:

1. **Overlay Behavior**: `CornerIndicator` wraps children and overlays them, whereas Spectrum indicators render inline beside content.
2. **Visual-Only Dot**: It renders a purely visual dot without a text label, whereas Spectrum components typically require or recommend visible text.
3. **Presence Signal**: Its purpose is a compact activity signal (e.g., indicating a tool is active or a change is pending) rather than a semantic status or category label.

Use `CornerIndicator` for non-intrusive activity signals on icons, thumbnails, or controls. For status labels visible alongside content, prefer `StatusLight`. For categorization metadata, prefer `Badge`.

## Variants

### Active State
When `isActive` is `true`, a small blue dot is rendered in the top-right corner. When `false`, the indicator is completely removed from the DOM.

## Implementation Details

### Structure
The component uses a Spectrum `View` with `position="relative"` to establish a coordinate system, and a `div` with `position="absolute"` for the dot.

### Styling with Tokens
The dot's size, position, and color are driven by Spectrum global dimension and color tokens:

| Property | Token | Purpose |
|----------|-------|---------|
| `top` / `right` | `--spectrum-global-dimension-size-50` | 4px offset from the corner. |
| `width` / `height` | `--spectrum-global-dimension-size-50` | 4px diameter dot. |
| `borderRadius` | `--spectrum-global-dimension-size-100` | Ensures the dot remains perfectly circular. |
| `backgroundColor` | `--spectrum-global-color-blue-700` | The Geti "Energy Blue" color. |

## Accessibility

- **Role**: The dot uses `role="status"` to identify itself as a status indicator to assistive technologies.
- **Label**: It includes `aria-label="Pending change"` by default. Note that in a production environment, this label should ideally be localized or context-specific.
- **Dynamic Updates**: Because it is conditionally rendered, screen readers will announce the presence of the status dot when it appears.
