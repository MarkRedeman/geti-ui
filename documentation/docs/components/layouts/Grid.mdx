# Grid

Grid is a CSS grid container that accepts Spectrum dimension tokens for column and row templates, gap, and alignment. Use it when you need a two-dimensional layout such as a card gallery, form with multiple columns, or structured dashboard panel.

```tsx
import { Grid } from '@geti-ai/ui';
```

- [React Spectrum Grid ↗](https://react-spectrum.adobe.com/react-spectrum/Grid.html)
- [Geti UI Theming](/components/theming)

## Examples

```tsx preview

export default function App() {
  return (
    <Grid columns="repeat(3, 1fr)" gap="size-200">
```

## Responsibility

`Grid` is a thin wrapper around React Spectrum `Grid`. It accepts Spectrum dimension tokens for sizing props and standard CSS grid values for `columns` and `rows`, enabling consistent spacing across the Geti design system.

## Variants and states

Grid has no visual variants — it is a layout primitive. Behaviour is fully controlled via props.

## Props (deep dive)

### Layout

- `columns`: CSS grid `grid-template-columns` value (e.g. `repeat(3, 1fr)`) or Spectrum token array.
- `rows`: CSS grid `grid-template-rows` value.
- `gap` / `rowGap` / `columnGap`: Spectrum dimension token or CSS value.
- `autoRows` / `autoColumns`: implicit track sizing.

### Alignment

- `justifyItems`: inline alignment within cells (`start | center | end | stretch`).
- `alignItems`: block alignment within cells (`start | center | end | stretch | baseline`).
- `justifyContent` / `alignContent`: distribution of tracks within the container.

### Sizing

- `width` / `height` / `minWidth` / `maxWidth`: accept Spectrum dimension tokens.

## Related components

- `Flex` — use for one-dimensional (row or column) layouts.
- `View` — use for a single boxed container without grid behaviour.
