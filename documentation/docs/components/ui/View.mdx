# View

View is a general-purpose layout container built on Spectrum style props. Use it when you need a div-like box with design-token-aware padding, background color, borders, or positioning — without writing raw CSS.

```tsx
import { View } from '@geti-ai/ui';
```

- [React Spectrum View ↗](https://react-spectrum.adobe.com/react-spectrum/View.html)
- [Geti UI Theming](/components/theming)

## Examples

### Padded container

```tsx preview
import { View } from '@geti-ai/ui';

export default function App() {
  return (
    <View padding="size-200" backgroundColor="gray-400" borderRadius="medium">
      Content inside a padded View
    </View>
  );
}
```

### With border

```tsx preview
import { View } from '@geti-ai/ui';

export default function App() {
  return (
    <View
      padding="size-200"
      borderWidth="thin"
      borderColor="blue-400"
      borderRadius="small"
    >
      Bordered view
    </View>
  );
}
```

### Relative positioning (for overlay children)

```tsx preview
import { View } from '@geti-ai/ui';

export default function App() {
  return (
    <View position="relative" width="size-1600" height="size-1600" backgroundColor="gray-300">
      <View position="absolute" top="size-0" right="size-0" width="size-200" height="size-200" backgroundColor="blue-500" borderRadius="large" />
    </View>
  );
}
```

## Responsibility

`View` is a thin wrapper around React Spectrum's `View`. All props pass through unchanged. The wrapper provides a consistent `@geti-ai/ui` import path.

## Variants and states

`View` has no variants or interactive states. It is purely a layout primitive.

## Props (deep dive)

### Layout

- `padding` / `paddingX` / `paddingY` / `paddingTop` etc.: Spectrum size tokens.
- `width` / `height` / `minWidth` / `maxWidth`: Spectrum dimension tokens or CSS.
- `position`: `'static' | 'relative' | 'absolute' | 'fixed' | 'sticky'`.
- `top` / `right` / `bottom` / `left`: positioning offsets when `position` is non-static.
- `zIndex`: stacking layer.
- `overflow`: `'hidden' | 'auto' | 'scroll' | 'visible'`.

### Visual

- `backgroundColor`: Spectrum color token (e.g. `'gray-800'`, `'blue-400'`).
- `borderRadius`: `'small' | 'medium' | 'large' | 'xlarge'`.
- `borderWidth`: `'thin' | 'thick' | 'thicker' | 'thickest'`.
- `borderColor`: Spectrum color token.

### Accessibility

- renders as a `<div>` by default,
- add `role` and `aria-label` when the container represents a meaningful region.

## Related components

- `Flex` — use for flex-layout containers with gap/alignment props.
- `Grid` — use for grid-layout containers.
- `Well` — use for visually inset content sections.
