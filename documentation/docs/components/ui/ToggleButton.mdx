# ToggleButton

ToggleButton is a button that maintains a selected/deselected state. Use it for persistent binary controls — show/hide a panel, lock/unlock an element, or enable/disable a mode.

```tsx
import { ToggleButton } from '@geti-ai/ui';
```

- [React Spectrum ToggleButton ↗](https://react-spectrum.adobe.com/react-spectrum/ToggleButton.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default (unselected)

```tsx preview
import { ToggleButton } from '@geti-ai/ui';

export default function App() {
  return <ToggleButton>Mute</ToggleButton>;
}
```

### Selected by default

```tsx preview
import { ToggleButton } from '@geti-ai/ui';

export default function App() {
  return <ToggleButton defaultSelected>Pinned</ToggleButton>;
}
```

### Emphasized

```tsx preview
import { ToggleButton } from '@geti-ai/ui';

export default function App() {
  return <ToggleButton isEmphasized defaultSelected>Active</ToggleButton>;
}
```

### Quiet

```tsx preview
import { ToggleButton } from '@geti-ai/ui';

export default function App() {
  return <ToggleButton isQuiet>Quiet</ToggleButton>;
}
```

### Disabled

```tsx preview
import { ToggleButton } from '@geti-ai/ui';

export default function App() {
  return <ToggleButton isDisabled>Locked</ToggleButton>;
}
```

## Responsibility

`ToggleButton` is a thin wrapper around React Spectrum's `ToggleButton`. All props pass through unchanged. The wrapper provides a consistent `@geti-ai/ui` import path and participates in Geti theming.

## Variants and states

| Prop | Default | Notes |
|---|---|---|
| `isEmphasized` | `false` | Accent color when selected (more prominent) |
| `isQuiet` | `false` | Minimal border at rest |
| `defaultSelected` | `false` | Uncontrolled initial selected state |
| `isSelected` | — | Controlled selected state |
| `isDisabled` | `false` | Non-interactive and visually muted |

## Props (deep dive)

### Content

- `children`: label or icon.

### Value / state

- `isSelected`: controlled boolean state.
- `defaultSelected`: uncontrolled initial state.

### Events

- `onChange`: called with new boolean state when selection changes.

### Visual

- `isEmphasized`: uses accent color to highlight selected state.
- `isQuiet`: minimal visual appearance.

### Accessibility

- Spectrum sets `aria-pressed` automatically,
- provide a clear label so selected vs unselected state is understandable out of context.
