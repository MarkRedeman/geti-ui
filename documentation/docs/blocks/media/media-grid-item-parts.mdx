# Media grid item parts

`@geti-ai/blocks` provides small, styled building blocks for `MediaGridItem` overlays:

- `MediaGridItemCheckbox`
- `MediaGridItemMenu`
- `MediaGridItemInfo`
- `MediaGridItemStatus`

Use these to keep corner UI consistent without repeating local styling.

```tsx
import {
  MediaGridItemCheckbox,
  MediaGridItemMenu,
  MediaGridItemInfo,
  MediaGridItemStatus,
} from '@geti-ai/blocks';
```

## `MediaGridItemCheckbox`

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `ariaLabel` | `string` | Yes | Accessible label for the checkbox. |
| `isSelected` | `boolean` | Yes | Checkbox selected state. |
| `onChange` | `(isSelected: boolean) => void` | Yes | Called with next selected state. |

```tsx preview
import { useState } from 'react';
import { ThemeProvider } from '@geti-ai/ui';
import { MediaGridItemCheckbox } from '@geti-ai/blocks';

export default function App() {
  const [selected, setSelected] = useState(true);

  return (
    <ThemeProvider>
      <MediaGridItemCheckbox
        ariaLabel="Select media item sample-1"
        isSelected={selected}
        onChange={() => setSelected((v) => !v)}
      />
    </ThemeProvider>
  );
}
```

## `MediaGridItemMenu`

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `ariaLabel` | `string` | No | Accessible label for the menu trigger. Default: `"Media actions"`. |
| `actions` | `MediaGridItemMenuAction[]` | Yes | Menu action list. |
| `onAction` | `(key: string) => void` | Yes | Called when an action is selected. |

`MediaGridItemMenuAction` shape:

| Field | Type | Description |
| --- | --- | --- |
| `key` | `string` | Stable action key passed to `onAction`. |
| `label` | `string` | Visible action label in the menu. |

```tsx preview
import { ThemeProvider } from '@geti-ai/ui';
import { MediaGridItemMenu } from '@geti-ai/blocks';

export default function App() {
  return (
    <ThemeProvider>
      <MediaGridItemMenu
        ariaLabel="Media actions"
        actions={[
          { key: 'annotate', label: 'Annotate' },
          { key: 'download', label: 'Download' },
          { key: 'delete', label: 'Delete' },
        ]}
        onAction={(key) => console.log('action', key)}
      />
    </ThemeProvider>
  );
}
```

## `MediaGridItemInfo`

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | Yes | Compact informational content (id, score, frame, etc.). |

```tsx preview
import { ThemeProvider } from '@geti-ai/ui';
import { MediaGridItemInfo } from '@geti-ai/blocks';

export default function App() {
  return (
    <ThemeProvider>
      <MediaGridItemInfo>frame #128</MediaGridItemInfo>
    </ThemeProvider>
  );
}
```

## `MediaGridItemStatus`

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `variant` | `'notice' \| 'positive' \| 'negative' \| 'neutral' \| 'info'` | Yes | Semantic status color. |
| `children` | `ReactNode` | Yes | Status label text. |

```tsx preview
import { ThemeProvider, Flex } from '@geti-ai/ui';
import { MediaGridItemStatus } from '@geti-ai/blocks';

export default function App() {
  return (
    <ThemeProvider>
      <Flex gap="size-200" wrap>
        <MediaGridItemStatus variant="notice">To review</MediaGridItemStatus>
        <MediaGridItemStatus variant="positive">Accepted</MediaGridItemStatus>
        <MediaGridItemStatus variant="negative">Rejected</MediaGridItemStatus>
        <MediaGridItemStatus variant="neutral">Neutral</MediaGridItemStatus>
        <MediaGridItemStatus variant="info">Info</MediaGridItemStatus>
      </Flex>
    </ThemeProvider>
  );
}
```
