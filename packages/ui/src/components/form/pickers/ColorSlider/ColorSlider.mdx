## Responsibility

`ColorSlider` lets users adjust a single channel of a color value (e.g. hue, alpha, red) using a horizontal or vertical slider. It wraps Spectrum's `ColorSlider`.

## Design

Pure thin wrapper:

```tsx
export interface ColorSliderProps extends SpectrumColorSliderProps {}
export const ColorSlider = (props: ColorSliderProps) => <SpectrumColorSlider {...props} />;
```

Key props: `value`, `onChange`, `channel` (e.g. `"hue"`, `"alpha"`, `"red"`), `orientation`. Spectrum renders a gradient track that reflects the chosen channel and a draggable thumb.

## Flow

Props in → forwarded to `SpectrumColorSlider` → user drags thumb → `onChange` fires with an updated `Color` value reflecting the new channel value.

## Integration

- **Depends on**: `@adobe/react-spectrum` (`ColorSlider`, `SpectrumColorSliderProps`)
- **Composed into**: `ColorPickerDialog` uses two `ColorSlider` instances — one for `"hue"`, one for `"alpha"`
- **Used alongside**: `ColorArea` (2D) and `ColorWheel` (hue ring)
