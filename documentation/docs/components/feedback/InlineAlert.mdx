# InlineAlert

`InlineAlert` displays a non-modal message associated with content in the current view. Use it to surface validation errors, warnings, or contextual info directly in the page rather than in a toast or dialog.

```tsx
import { InlineAlert } from '@geti/ui';
```

- [React Spectrum InlineAlert ↗](https://react-spectrum.adobe.com/react-spectrum/InlineAlert.html)
- [Geti UI Theming](/components/theming)

## Examples

### Neutral — general information

```tsx preview
import { InlineAlert } from '@geti/ui';
import { Heading, Content } from '@adobe/react-spectrum';

export default function App() {
  return (
    <InlineAlert variant="neutral">
      <Heading>Note</Heading>
      <Content>This dataset does not have any annotations yet.</Content>
    </InlineAlert>
  );
}
```

## Responsibility

`InlineAlert` is a thin wrapper around React Spectrum's `InlineAlert`. It passes all props through unchanged. The component requires `Heading` and `Content` from `@adobe/react-spectrum` as children to compose the alert structure.

## Variants and states

| Variant | When to use |
|---|---|
| `neutral` | Supplementary context with no urgency |
| `info` | Informational message about system state |
| `positive` | Confirmation that an action succeeded |
| `notice` | Warning — something may go wrong |
| `negative` | Error — something has gone wrong |

## Props (deep dive)

### Content

- `children`: compose with `<Heading>` for the alert title and `<Content>` for the body text. Both are from `@adobe/react-spectrum`.
- `variant`: one of `neutral | info | positive | notice | negative`.

### Accessibility

- The alert is rendered with `role="alert"` semantics by Spectrum so screen readers announce it.
- Use specific, actionable heading text (`"Missing required field"`) over generic phrases (`"Error"`).
- `InlineAlert` is static — it does not dismiss. For dismissible messages, use `Toast`.

## Related components

- `Toast` — use for transient success/error notifications that auto-dismiss.
- `Badge` — use for compact status labels without a heading or body text.
- `StatusLight` — use for inline coloured dots in data rows.
