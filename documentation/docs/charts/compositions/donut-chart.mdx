# Donut chart (composition)

`DonutChart` is a pie-style composition with a hollow center, useful for share-of-total summaries and score distributions.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | - | Array of data objects with name/value keys |
| `valueKey` | `string` | `'value'` | Key in each data object for the numeric value |
| `nameKey` | `string` | `'name'` | Key in each data object for the slice label |
| `colors` | `string[]` | theme colors | Array of fill colors for slices |
| `innerRadius` | `string \| number` | `'58%'` | Inner radius of the donut hole |
| `outerRadius` | `string \| number` | `'82%'` | Outer radius of the ring |
| `paddingAngle` | `number` | `2` | Gap in degrees between slices |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLabels` | `boolean` | `false` | Render percentage labels on slices |
| `animate` | `boolean` | `false` | Enable entry animation |
| `height` | `number` | `300` | Chart height in pixels |

## Variant - default

```tsx preview
import { DonutChart } from '@geti-ui/charts';

const data = [
  { name: 'Images', value: 52 },
  { name: 'Videos', value: 18 },
  { name: 'Annotations', value: 22 },
  { name: 'Models', value: 8 },
];

export default function App() {
  return <DonutChart data={data} aria-label="Dataset storage share" height={320} />;
}
```

## Variant - labels on

```tsx preview
import { DonutChart } from '@geti-ui/charts';

const data = [
  { name: 'Car', value: 4200 },
  { name: 'Person', value: 5600 },
  { name: 'Bicycle', value: 2100 },
  { name: 'Bus', value: 1400 },
];

export default function App() {
  return <DonutChart data={data} showLabels aria-label="Class share with labels" height={320} />;
}
```

## Variant - no legend, custom ring thickness

```tsx preview
import { DonutChart } from '@geti-ui/charts';

const data = [
  { name: 'Accepted', value: 71 },
  { name: 'Rejected', value: 19 },
  { name: 'Needs review', value: 10 },
];

export default function App() {
  return (
    <DonutChart
      data={data}
      showLegend={false}
      innerRadius="68%"
      outerRadius="86%"
      colors={['#22c55e', '#ef4444', '#f59e0b']}
      aria-label="Review status donut"
      height={300}
    />
  );
}
```

## Variant - train / validation / test palette

```tsx preview
import { DonutChart, getDatasetSubsetColor } from '@geti-ui/charts';

const data = [
  { name: 'Train', value: 70 },
  { name: 'Validation', value: 20 },
  { name: 'Test', value: 10 },
];

export default function App() {
  return (
    <DonutChart
      data={data}
      colors={[
        getDatasetSubsetColor('train')!,
        getDatasetSubsetColor('validation')!,
        getDatasetSubsetColor('test')!,
      ]}
      showLabels
      aria-label="Dataset subset donut"
      height={320}
    />
  );
}
```
