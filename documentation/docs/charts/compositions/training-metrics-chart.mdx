# Training metrics chart (composition)

`TrainingMetricsChart` is an ML-focused composition for train/validation/test metric timelines.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `object[]` | required | Timeline rows (e.g. epochs) |
| `metrics` | `TrainingMetricSeriesConfig[]` | required | Metric groups with `trainKey` and optional `validationKey`/`testKey` |
| `xAxisKey` | `string` | `'epoch'` | X-axis key |
| `showLegend` | `boolean` | `true` | Show legend |
| `showTooltip` | `boolean` | `true` | Show tooltip |
| `animate` | `boolean` | `false` | Opt-in animation |
| `xScale` / `yScale` | `AxisScaleConfig` | - | Axis scale settings (e.g. log) |
| `xAxisProps` / `yAxisProps` | `XAxisProps` / `YAxisProps` | - | Axis passthrough props |

## Variant - loss + mAP over epochs

```tsx preview
import { TrainingMetricsChart } from '@geti-ui/charts';

const data = [
  { epoch: 1, trainLoss: 1.42, valLoss: 1.53, trainMap: 0.21, valMap: 0.18 },
  { epoch: 2, trainLoss: 1.08, valLoss: 1.20, trainMap: 0.34, valMap: 0.30 },
  { epoch: 3, trainLoss: 0.86, valLoss: 0.98, trainMap: 0.48, valMap: 0.43 },
  { epoch: 4, trainLoss: 0.70, valLoss: 0.83, trainMap: 0.59, valMap: 0.54 },
  { epoch: 5, trainLoss: 0.58, valLoss: 0.72, trainMap: 0.67, valMap: 0.62 },
  { epoch: 6, trainLoss: 0.49, valLoss: 0.64, trainMap: 0.74, valMap: 0.69 },
];

export default function App() {
  return (
    <TrainingMetricsChart
      data={data}
      xAxisKey="epoch"
      metrics={[
        { name: 'Loss', trainKey: 'trainLoss', validationKey: 'valLoss' },
        { name: 'mAP', trainKey: 'trainMap', validationKey: 'valMap' },
      ]}
      aria-label="Training and validation metrics over epochs"
      height={340}
    />
  );
}
```

## Variant - include test curve + log scale

```tsx preview
import { TrainingMetricsChart } from '@geti-ui/charts';

const data = [
  { epoch: 1, trainLoss: 2.31, valLoss: 2.54, testLoss: 2.71 },
  { epoch: 2, trainLoss: 1.20, valLoss: 1.41, testLoss: 1.56 },
  { epoch: 3, trainLoss: 0.63, valLoss: 0.74, testLoss: 0.86 },
  { epoch: 4, trainLoss: 0.31, valLoss: 0.39, testLoss: 0.47 },
  { epoch: 5, trainLoss: 0.16, valLoss: 0.21, testLoss: 0.26 },
  { epoch: 6, trainLoss: 0.08, valLoss: 0.12, testLoss: 0.15 },
];

export default function App() {
  return (
    <TrainingMetricsChart
      data={data}
      xAxisKey="epoch"
      yScale={{ scale: 'log', domain: [0.05, 'auto'] }}
      metrics={[
        {
          name: 'Loss',
          trainKey: 'trainLoss',
          validationKey: 'valLoss',
          testKey: 'testLoss',
        },
      ]}
      aria-label="Train/validation/test loss on logarithmic scale"
      height={320}
    />
  );
}
```
