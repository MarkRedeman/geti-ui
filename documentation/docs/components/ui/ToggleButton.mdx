# ToggleButton

ToggleButton is a button that maintains a selected/deselected state. Use it for persistent binary controls ‚Äî show/hide a panel, lock/unlock an element, or enable/disable a mode.

```tsx
import { ToggleButton } from '@geti/ui';
```

## Responsibility

`ToggleButton` is a thin wrapper around React Spectrum's `ToggleButton`. All props pass through unchanged. The wrapper provides a consistent `@geti/ui` import path and participates in Geti theming.

## Variants and states

| Prop | Default | Notes |
|---|---|---|
| `isEmphasized` | `false` | Accent color when selected (more prominent) |
| `isQuiet` | `false` | Minimal border at rest |
| `defaultSelected` | `false` | Uncontrolled initial selected state |
| `isSelected` | ‚Äî | Controlled selected state |
| `isDisabled` | `false` | Non-interactive and visually muted |

## Props (deep dive)

### Content

- `children`: label or icon.

### Value / state

- `isSelected`: controlled boolean state.
- `defaultSelected`: uncontrolled initial state.

### Events

- `onChange`: called with new boolean state when selection changes.

### Visual

- `isEmphasized`: uses accent color to highlight selected state.
- `isQuiet`: minimal visual appearance.

### Accessibility

- Spectrum sets `aria-pressed` automatically,
- provide a clear label so selected vs unselected state is understandable out of context.

## Related components

- `ToggleButtons` ‚Äî use for mutually exclusive segmented selections (only one can be active).
- `ActionButton` ‚Äî use for one-shot actions without persistent state.
- `Switch` ‚Äî use for on/off settings that feel more like a preference than an action.
- `Checkbox` ‚Äî use when the selected state is part of a form submission.
