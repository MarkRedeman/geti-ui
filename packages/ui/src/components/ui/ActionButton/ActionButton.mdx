ActionButtons allow users to perform an action. They’re used for similar, task-based options within a workflow, and are ideal for interfaces where buttons aren’t meant to draw a lot of attention.

## Import

```tsx
import { ActionButton } from '@geti/ui';
```

## Reference

[Adobe React Spectrum ActionButton](https://react-spectrum.adobe.com/react-spectrum/ActionButton.html)

## Variants

### Geti Color Variants

The Geti `ActionButton` extends Spectrum's base with a `colorVariant` prop for specific product themes:

| Variant | Description |
|---|---|
| `dark` (default) | Subtle border on hover/active. Standard Geti look for dark panels/toolbars. |
| `light` | Blue-tinted hover/active. Optimized for light header areas. |
| `blue` | Transparent background with blue text. Ideal for icon-only, transparent contexts. |

### Visual Options

Following Spectrum's core options:

- **Quiet**: Use `isQuiet` for a button with no background or border until hover.
- **Disabled**: Use `isDisabled` to prevent user interaction.
- **Static Color**: Use `staticColor="white" | "black"` when placed over colored backgrounds.

## Props

### Button Props

| Name | Type | Default | Description |
|---|---|---|---|
| `colorVariant` | `'dark' \| 'light' \| 'blue'` | `'dark'` | Applies a Geti-specific visual style. |
| `isQuiet` | `boolean` | `false` | Whether the button should be displayed with a quiet style. |
| `isDisabled` | `boolean` | `false` | Whether the button is disabled. |
| `staticColor` | `'white' \| 'black'` | — | The static color style to apply over colored backgrounds. |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | The behavior of the button when used in an HTML form. |

### Content

| Name | Type | Default | Description |
|---|---|---|---|
| `children` | `ReactNode` | — | The content to display in the button (Label, Icon, or both). |

### Events

| Name | Type | Description |
|---|---|---|
| `onPress` | `(e: PressEvent) => void` | Handler that is called when the press is released over the target. |
| `onPressStart` | `(e: PressEvent) => void` | Handler that is called when a press interaction starts. |
| `onPressEnd` | `(e: PressEvent) => void` | Handler that is called when a press interaction ends. |
| `onFocus` | `(e: FocusEvent) => void` | Handler that is called when the element receives focus. |
| `onBlur` | `(e: FocusEvent) => void` | Handler that is called when the element loses focus. |

---

## Responsibility

The `ActionButton` wraps Adobe React Spectrum's `ActionButton`. Its primary responsibility is to:
1. Provide Geti-specific color variants (`dark`, `light`, `blue`) via the `colorVariant` prop.
2. Ensure stable styling by merging `UNSAFE_className` with internal Geti CSS modules.
