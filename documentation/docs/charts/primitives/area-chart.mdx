# Area chart (primitive)

Use Recharts `AreaChart` with Geti primitives for custom fills and layered styling.

## Props

Key Recharts props used on the `Area` primitive:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | required | Key in each data object to plot |
| `type` | `string` | `'linear'` | Curve interpolation (e.g. `'monotone'`, `'linear'`, `'step'`) |
| `stroke` | `string` | — | Line/border color |
| `fill` | `string` | — | Fill color or gradient reference (e.g. `url(#id)`) |
| `fillOpacity` | `number` | `0.5` | Opacity of the filled area (0–1) |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |
| `name` | `string` | — | Series label shown in tooltip and legend |

## Variant — solid fill (no fade)

```tsx preview
import { Area, AreaChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <AreaChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="step" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend />
        <Area type="monotone" dataKey="train" stroke="var(--spectrum-global-color-blue-500)" fill="var(--spectrum-global-color-blue-500)" fillOpacity={0.2} isAnimationActive={false} />
        <Area type="monotone" dataKey="val" stroke="var(--spectrum-global-color-green-500)" fill="var(--spectrum-global-color-green-500)" fillOpacity={0.2} isAnimationActive={false} />
      </AreaChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — fade gradient fill

```tsx preview
import { Area, AreaChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <AreaChart data={data}>
        <defs>
          <linearGradient id="trainGrad" x1="0" y1="0" x2="0" y2="1">
            <stop offset="5%" stopColor="var(--spectrum-global-color-blue-500)" stopOpacity={0.3} />
            <stop offset="95%" stopColor="var(--spectrum-global-color-blue-500)" stopOpacity={0} />
          </linearGradient>
          <linearGradient id="valGrad" x1="0" y1="0" x2="0" y2="1">
            <stop offset="5%" stopColor="var(--spectrum-global-color-green-500)" stopOpacity={0.3} />
            <stop offset="95%" stopColor="var(--spectrum-global-color-green-500)" stopOpacity={0} />
          </linearGradient>
        </defs>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="step" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend />
        <Area type="monotone" dataKey="train" stroke="var(--spectrum-global-color-blue-500)" fill="url(#trainGrad)" isAnimationActive={false} />
        <Area type="monotone" dataKey="val" stroke="var(--spectrum-global-color-green-500)" fill="url(#valGrad)" isAnimationActive={false} />
      </AreaChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — time-based area chart

```tsx preview
import { Area, AreaChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { ts: Date.parse('2026-01-01T00:00:00Z'), value: 0.42 },
  { ts: Date.parse('2026-01-02T00:00:00Z'), value: 0.55 },
  { ts: Date.parse('2026-01-03T00:00:00Z'), value: 0.63 },
  { ts: Date.parse('2026-01-04T00:00:00Z'), value: 0.58 },
  { ts: Date.parse('2026-01-05T00:00:00Z'), value: 0.71 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <AreaChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis
          dataKey="ts"
          type="number"
          scale="time"
          domain={['dataMin', 'dataMax']}
          tickFormatter={(v) => new Date(Number(v)).toLocaleDateString('en-US', { month: 'short', day: 'numeric' })}
        />
        <GetiYAxis />
        <GetiTooltip />
        <Area type="monotone" dataKey="value" stroke="var(--spectrum-global-color-blue-500)" fill="var(--spectrum-global-color-blue-500)" fillOpacity={0.2} isAnimationActive={false} />
      </AreaChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — log scale on Y axis

```tsx preview
import { Area, AreaChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { batch: 1, error: 600 },
  { batch: 2, error: 240 },
  { batch: 3, error: 95 },
  { batch: 4, error: 36 },
  { batch: 5, error: 15 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <AreaChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="batch" />
        <GetiYAxis scale="log" domain={[1, 'auto']} />
        <GetiTooltip />
        <Area type="monotone" dataKey="error" stroke="var(--spectrum-global-color-blue-500)" fill="var(--spectrum-global-color-blue-500)" fillOpacity={0.2} isAnimationActive={false} />
      </AreaChart>
    </GetiResponsiveContainer>
  );
}
```
