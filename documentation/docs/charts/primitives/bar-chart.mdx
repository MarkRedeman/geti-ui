# Bar chart (primitive)

Use Recharts `BarChart` primitives when you need full control over stack groups and mixed layers.

## Props

Key Recharts props used on the `Bar` and `BarChart` primitives:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | required | Key in each data object to plot |
| `fill` | `string` | — | Bar fill color |
| `stackId` | `string` | — | Bars sharing the same `stackId` are stacked |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |
| `name` | `string` | — | Series label shown in tooltip and legend |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | Orientation of the bar chart |

## Variant — grouped bars (`stackId` off)

```tsx preview
import { Bar, BarChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { model: 'MobileNet', precision: 84, recall: 80 },
  { model: 'ResNet-50', precision: 89, recall: 87 },
  { model: 'EfficientDet', precision: 91, recall: 90 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <BarChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="model" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend />
        <Bar dataKey="precision" fill="var(--spectrum-global-color-blue-500)" isAnimationActive={false} />
        <Bar dataKey="recall" fill="var(--spectrum-global-color-green-500)" isAnimationActive={false} />
      </BarChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — stacked bars (`stackId` on)

```tsx preview
import { Bar, BarChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { label: 'Annotated', ok: 900, review: 60, rejected: 40 },
  { label: 'Total', ok: 900, review: 60, rejected: 40 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <BarChart data={data}>
        <GetiCartesianGrid />
        <GetiXAxis dataKey="label" />
        <GetiYAxis />
        <GetiTooltip />
        <GetiLegend />
        <Bar dataKey="ok" stackId="stack" fill="var(--spectrum-global-color-green-500)" isAnimationActive={false} />
        <Bar dataKey="review" stackId="stack" fill="var(--spectrum-global-color-yellow-500)" isAnimationActive={false} />
        <Bar dataKey="rejected" stackId="stack" fill="var(--spectrum-global-color-red-500)" isAnimationActive={false} />
      </BarChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — horizontal bars (`layout="vertical"`)

```tsx preview
import { Bar, BarChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { label: 'Background cleanup', seconds: 12 },
  { label: 'Augmentation', seconds: 28 },
  { label: 'Training', seconds: 140 },
  { label: 'Validation', seconds: 34 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={280}>
      <BarChart data={data} layout="vertical" margin={{ top: 8, right: 16, bottom: 8, left: 60 }}>
        <GetiCartesianGrid />
        <GetiXAxis type="number" />
        <GetiYAxis type="category" dataKey="label" width={120} />
        <GetiTooltip />
        <Bar dataKey="seconds" fill="var(--spectrum-global-color-blue-500)" isAnimationActive={false} />
      </BarChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — log scale on value axis

```tsx preview
import { Bar, BarChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ai/charts/primitives';

const data = [
  { stage: 'raw', count: 12 },
  { stage: 'filtered', count: 140 },
  { stage: 'augmented', count: 2300 },
  { stage: 'final', count: 10500 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={260}>
      <BarChart data={data} layout="horizontal">
        <GetiCartesianGrid />
        <GetiXAxis dataKey="stage" />
        <GetiYAxis scale="log" domain={[1, 'auto']} />
        <GetiTooltip />
        <Bar dataKey="count" fill="var(--spectrum-global-color-blue-500)" isAnimationActive={false} />
      </BarChart>
    </GetiResponsiveContainer>
  );
}
```
