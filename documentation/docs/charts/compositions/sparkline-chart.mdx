# Sparkline chart (composition)

`Sparkline` is a compact, axis-free trend chart for inline usage.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | ŌĆö | Array of data objects with a numeric value per point |
| `dataKey` | `string` | `'value'` | Key in each data object that holds the value to plot |
| `width` | `CSSProperties['width']` | `'100%'` | Width of the sparkline |
| `height` | `number` | `40` | Height of the sparkline in pixels |
| `color` | `string` | theme color | Stroke color override |
| `strokeWidth` | `number` | `1.5` | Line stroke width |
| `area` | `boolean` | `false` | Fill the area under the line |
| `fillOpacity` | `number` | `0.15` | Opacity of the fill area (0ŌĆō1) |
| `curve` | `SparklineCurve` | `'monotone'` | Curve interpolation type |
| `animate` | `boolean` | `false` | Enable entry animation |
| `aria-label` | `string` | ŌĆö | Accessible label for screen readers |

## Variant ŌĆö `area` off

```tsx preview
import { Sparkline } from '@geti-ui/charts';

export default function App() {
  return <Sparkline data={[72, 75, 79, 81, 83, 85, 88].map((value) => ({ value }))} area={false} />;
}
```

## Variant ŌĆö `area` on

```tsx preview
import { Sparkline } from '@geti-ui/charts';

export default function App() {
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
      <Sparkline data={[72, 75, 79, 81, 83, 85, 88].map((value) => ({ value }))} area />
      <Sparkline data={[80, 82, 84, 86, 87, 89, 91].map((value) => ({ value }))} />
    </div>
  );
}
```

## Variant ŌĆö categorical mini-trends

```tsx preview
import { Sparkline } from '@geti-ui/charts';

const trends = {
  car: [51, 52, 53, 54, 55, 56, 56],
  person: [63, 62, 61, 63, 64, 65, 66],
  bike: [44, 45, 44, 47, 46, 48, 49],
};

export default function App() {
  return (
    <div style={{ display: 'grid', gap: 10 }}>
      {Object.entries(trends).map(([label, values]) => (
        <div key={label} style={{ display: 'flex', alignItems: 'center', gap: 10 }}>
          <span style={{ width: 70, textTransform: 'capitalize' }}>{label}</span>
          <Sparkline data={values.map((value) => ({ value }))} area />
        </div>
      ))}
    </div>
  );
}
```
