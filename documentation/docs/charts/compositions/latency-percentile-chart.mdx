# Latency percentile chart (composition)

`LatencyPercentileChart` visualizes P50/P95/P99 latency trends across runs or batch sizes.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of data objects, one per run or batch size |
| `xAxisKey` | `string` | `'label'` | Key in each data object for the X (category) axis |
| `p50Key` | `string` | `'p50'` | Key in each data object for the P50 latency value |
| `p95Key` | `string` | `'p95'` | Key in each data object for the P95 latency value |
| `p99Key` | `string` | `'p99'` | Key in each data object for the P99 latency value |
| `unit` | `string` | `'ms'` | Unit suffix appended to tooltip values |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `targetLatency` | `number` | — | Draw a horizontal reference line at this latency value |

## Variant — latency percentiles by batch size

```tsx preview
import { LatencyPercentileChart } from '@geti-ai/charts';

const data = [
  { label: 'B1', p50: 14.2, p95: 22.5, p99: 31.8 },
  { label: 'B2', p50: 16.8, p95: 26.1, p99: 35.7 },
  { label: 'B4', p50: 22.3, p95: 33.4, p99: 44.6 },
  { label: 'B8', p50: 31.4, p95: 45.8, p99: 61.2 },
  { label: 'B16', p50: 47.7, p95: 66.9, p99: 88.4 },
];

export default function App() {
  return (
    <LatencyPercentileChart
      data={data}
      aria-label="Latency percentiles by batch size"
      height={360}
    />
  );
}
```

## Variant — with target latency reference

```tsx preview
import { LatencyPercentileChart } from '@geti-ai/charts';

const data = [
  { label: 'Run A', p50: 18.4, p95: 27.8, p99: 39.6 },
  { label: 'Run B', p50: 16.9, p95: 25.1, p99: 35.2 },
  { label: 'Run C', p50: 15.7, p95: 23.2, p99: 31.7 },
  { label: 'Run D', p50: 14.8, p95: 21.9, p99: 29.6 },
];

export default function App() {
  return (
    <LatencyPercentileChart
      data={data}
      targetLatency={30}
      unit="ms"
      aria-label="Latency percentiles with target threshold"
      height={360}
    />
  );
}
```
