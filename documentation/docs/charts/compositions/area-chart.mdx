# Area chart (composition)

`AreaChart` is a ready-to-use composition with themed defaults, optional stacking, and optional fade gradients.

## Variant ‚Äî `stacked` and `fade` off

```tsx preview
import { AreaChart } from '@geti-ai/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      series={[
        { dataKey: 'train', name: 'Train IoU', stacked: false, fade: false },
        { dataKey: 'val', name: 'Val IoU', stacked: false, fade: false },
      ]}
      aria-label="IoU unstacked no fade"
      height={260}
    />
  );
}
```

## Variant ‚Äî `stacked` and `fade` on

```tsx preview
import { AreaChart } from '@geti-ai/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      series={[
        { dataKey: 'train', name: 'Train IoU', stacked: true, fade: true },
        { dataKey: 'val', name: 'Val IoU', stacked: true, fade: true },
      ]}
      aria-label="IoU over training steps"
      height={260}
    />
  );
}
```

`series[].fade` uses a vertical gradient fill (strong near the line, fading to transparent at the bottom), matching the custom-chart style.

## Variant ‚Äî chart-level booleans on (`showLegend`, `animate`)

```tsx preview
import { AreaChart } from '@geti-ai/charts';

const data = [
  { step: 1, train: 0.45, val: 0.41 },
  { step: 2, train: 0.51, val: 0.46 },
  { step: 3, train: 0.62, val: 0.57 },
  { step: 4, train: 0.71, val: 0.65 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="step"
      showLegend
      animate
      series={[
        { dataKey: 'train', name: 'Train IoU', fade: true },
        { dataKey: 'val', name: 'Val IoU', fade: true },
      ]}
      aria-label="IoU with legend and animation"
      height={260}
    />
  );
}
```

## Variant ‚Äî time-based X axis

```tsx preview
import { AreaChart } from '@geti-ai/charts';

const data = [
  { ts: Date.parse('2026-01-01T00:00:00Z'), utilization: 0.42 },
  { ts: Date.parse('2026-01-02T00:00:00Z'), utilization: 0.55 },
  { ts: Date.parse('2026-01-03T00:00:00Z'), utilization: 0.63 },
  { ts: Date.parse('2026-01-04T00:00:00Z'), utilization: 0.58 },
  { ts: Date.parse('2026-01-05T00:00:00Z'), utilization: 0.71 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="ts"
      xScale={{ scale: 'time' }}
      xAxisProps={{
        type: 'number',
        domain: ['dataMin', 'dataMax'],
        tickFormatter: (v) => new Date(Number(v)).toLocaleDateString('en-US', { month: 'short', day: 'numeric' }),
      }}
      series={[{ dataKey: 'utilization', name: 'GPU utilization', fade: true }]}
      aria-label="Utilization over time"
      height={260}
    />
  );
}
```

## Variant ‚Äî log-scaled Y axis

```tsx preview
import { AreaChart } from '@geti-ai/charts';

const data = [
  { batch: 1, error: 600 },
  { batch: 2, error: 240 },
  { batch: 3, error: 95 },
  { batch: 4, error: 36 },
  { batch: 5, error: 15 },
];

export default function App() {
  return (
    <AreaChart
      data={data}
      xAxisKey="batch"
      yScale={{ scale: 'log', domain: [1, 'auto'] }}
      series={[{ dataKey: 'error', name: 'Error', fade: true }]}
      aria-label="Log-scale area chart"
      height={260}
    />
  );
}
```
