# Heading

`Heading` renders section and page titles with Spectrum heading semantics.

```tsx
import { Heading } from '@geti-ai/ui';
```

- [React Spectrum Heading](https://react-spectrum.adobe.com/react-spectrum/Heading.html)

<Stories />

## Responsibility

`Heading` is a thin wrapper around React Spectrum `Heading`. It keeps typography primitives available via `@geti-ai/ui` and aligned with the active Geti theme.

## Variants and states

`Heading` has no wrapper-specific variants. Visual weight and scale are controlled by Spectrum and context.

## Props

<Controls of={DocStories.Default} />

### Content

- `children`: heading text or inline markup.

### Semantics

- `level`: heading level (`1`-`6`) used for semantic hierarchy and accessibility.
- Choose levels by document outline, not by desired visual size.

### Accessibility

- Use heading levels in logical order for document structure.
- Prefer meaningful labels; avoid decorative-only headings.

## Related components

- `Text` for body copy and supporting text.
- `View` for section containers.
- `Flex` for heading + actions layouts.
