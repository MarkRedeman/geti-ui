# Flex

Flex is a CSS flexbox container that accepts Spectrum dimension tokens for `gap`, `width`, `height`, and other spacing values. Use it to build horizontal or vertical layouts with consistent Spectrum-aligned spacing.

```tsx
import { Flex } from '@geti-ai/ui';
```

## Examples

### Row with gap

```tsx preview
import { Flex } from '@geti-ai/ui';

export default function App() {
  return (
    <Flex direction="row" gap="size-200">
      <div style={{ background: '#3b82f6', padding: '8px 16px', borderRadius: 4 }}>Item 1</div>
      <div style={{ background: '#3b82f6', padding: '8px 16px', borderRadius: 4 }}>Item 2</div>
      <div style={{ background: '#3b82f6', padding: '8px 16px', borderRadius: 4 }}>Item 3</div>
    </Flex>
  );
}
```

### Centered alignment

```tsx preview

  return (
    <Flex direction="row" gap="size-200" alignItems="center" justifyContent="center">

```

## Responsibility

`Flex` is a thin wrapper around React Spectrum `Flex`. It accepts Spectrum dimension tokens (`size-100`, `size-200`, etc.) for spacing props rather than raw CSS values, keeping layouts consistent with the Geti design token system.

## Variants and states

Flex has no visual variants â€” it is a layout primitive. Behaviour is controlled via props.

## Props (deep dive)

### Layout

- `direction`: `row` (default) | `column` | `row-reverse` | `column-reverse`.
- `gap`: Spectrum dimension token or CSS value for spacing between children.
- `wrap`: allows items to wrap onto multiple lines.

### Alignment

- `alignItems`: cross-axis alignment (`start | center | end | stretch | baseline`).
- `justifyContent`: main-axis distribution (`start | center | end | space-between | space-around | space-evenly`).
- `alignContent`: multi-line cross-axis alignment.

### Sizing

- `width` / `height` / `minWidth` / `maxWidth`: accept Spectrum dimension tokens.
