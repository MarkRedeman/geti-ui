# Bar chart (composition)

`BarChart` is a ready-to-use composition for grouped or stacked categorical comparisons.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of data objects for the chart |
| `series` | `BarSeriesConfig[]` | — | Series definitions; each entry has `dataKey`, `name`, `color?` |
| `xAxisKey` | `string` | — | Key in each data object for the X (category) axis |
| `stacked` | `boolean` | `false` | Stack bars on top of each other |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | `'horizontal'` = vertical bars, `'vertical'` = horizontal bars |
| `height` | `number` | `300` | Chart height in pixels |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `false` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `xScale` | `ScaleConfig` | — | X axis scale config |
| `yScale` | `ScaleConfig` | — | Y axis scale config (e.g. `{ scale: 'log', domain: [1, 'auto'] }`) |
| `xAxisProps` | `XAxisProps` | — | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | — | Extra props forwarded to the Recharts `YAxis` |

## Variant — `stacked` off (grouped)

```tsx preview
import { BarChart } from '@geti-ai/charts';

const data = [
  { model: 'MobileNet', precision: 84, recall: 80 },
  { model: 'ResNet-50', precision: 89, recall: 87 },
  { model: 'EfficientDet', precision: 91, recall: 90 },
];

export default function App() {
  return (
    <BarChart
      data={data}
      xAxisKey="model"
      stacked={false}
      series={[
        { dataKey: 'precision', name: 'Precision (%)' },
        { dataKey: 'recall', name: 'Recall (%)' },
      ]}
      aria-label="Grouped bar chart"
      height={260}
    />
  );
}
```

## Variant — train / validation / test palette

```tsx preview
import { BarChart, withDatasetSubsetPalette } from '@geti-ai/charts';

const data = [
  { split: 'Fold A', train: 0.91, val: 0.85, test: 0.83 },
  { split: 'Fold B', train: 0.93, val: 0.86, test: 0.84 },
  { split: 'Fold C', train: 0.92, val: 0.87, test: 0.85 },
];

export default function App() {
  const series = withDatasetSubsetPalette([
    { dataKey: 'train', name: 'Training' },
    { dataKey: 'val', name: 'Validation' },
    { dataKey: 'test', name: 'Test' },
  ]);

  return (
    <BarChart
      data={data}
      xAxisKey="split"
      showLegend
      series={series}
      aria-label="Training/validation/test metric by split"
      height={260}
    />
  );
}
```
