# Assets

`@geti-ai/ui` ships production-ready SVG icons, illustration images, domain visuals, and primary-tool thumbnails as static assets.

## Subpath imports

Each asset group is available as a standalone subpath so consumers only pay for what they use:

| Subpath | Contents |
|---|---|
| `@geti-ai/ui/icons` | Custom SVG icons + Adobe Spectrum workflow re-exports |
| `@geti-ai/ui/assets/images` | Illustration SVGs and loading/background images (webp) |
| `@geti-ai/ui/assets/domains` | Domain-type SVG thumbnails (classification, detection, …) |
| `@geti-ai/ui/assets/primary-tools` | Annotation tool preview images (webp) |
| `@geti-ai/ui/assets` | Namespace re-export of all the above |

## Quick start

```tsx
// SVG icon rendered as a React component
import { Add, Delete } from '@geti-ai/ui/icons';

// Illustration image
import { WorkInProgressIcon } from '@geti-ai/ui/assets/images';

// webp URL (use in <img src={…}>)
import { LoginBackground } from '@geti-ai/ui/assets/images';

// Domain thumbnail
import { DetectionImg } from '@geti-ai/ui/assets/domains';

// Tool thumbnail URL
import { BrushTool } from '@geti-ai/ui/assets/primary-tools';
```
