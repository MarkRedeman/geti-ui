# Managed tabs

Use `ManagedTab` when a selected tab should expose contextual actions (rename, export, delete, etc.) directly in the tab header.

Typical application flow:

1. Your app owns tab data and selected key.
2. `ManagedTab` renders each tab label.
3. The selected tab opens a menu with actions for that dataset/project.
4. Your app handles action side effects (state updates, API calls, confirmation dialogs).

```tsx
import { ManagedTab, type ManagedTabAction } from '@geti-ai/blocks';
```

## In-app dataset management example

```tsx preview
import { useState } from 'react';
import { ManagedTab, type ManagedTabAction } from '@geti-ai/blocks';
import { ActionButton, TabItem, TabList, TabPanels, Tabs, Text, ThemeProvider, View } from '@geti-ai/ui';
import { Add } from '@geti-ai/ui/icons';
import { Icon } from '@adobe/react-spectrum';

type Dataset = {
  id: string;
  name: string;
  images: number;
};

const ACTIONS: ManagedTabAction[] = [
  { key: 'edit', label: 'Edit dataset name' },
  { key: 'delete', label: 'Delete dataset' },
  { key: 'export', label: 'Export dataset' },
];

const initialDatasets: Dataset[] = [
  { id: 'train', name: 'Training set', images: 1240 },
  { id: 'val', name: 'Validation set', images: 340 },
  { id: 'test', name: 'Testing set', images: 510 },
  { id: 'night', name: 'Night shift', images: 188 },
  { id: 'drift', name: 'Drift watch', images: 72 },
];

const createDataset = (index: number): Dataset => ({
  id: `dataset-${index + 1}`,
  name: `Dataset ${index + 1}`,
  images: 100 + (index + 1) * 25,
});

export default function App() {
  const [datasets, setDatasets] = useState(initialDatasets);
  const [selectedId, setSelectedId] = useState(initialDatasets[0].id);
  const [lastAction, setLastAction] = useState('No action yet.');

  const onItemAction = (action: string, dataset: Dataset) => {
    if (action === 'delete') {
      if (datasets.length <= 1) {
        setLastAction('Cannot delete the only remaining dataset.');
        return;
      }

      const next = datasets.filter((item) => item.id !== dataset.id);
      setDatasets(next);
      setSelectedId(next[0].id);
      setLastAction(`Deleted dataset "${dataset.name}".`);
      return;
    }

    if (action === 'edit') {
      setDatasets((current) =>
        current.map((item) =>
          item.id === dataset.id && !item.name.endsWith(' (edited)')
            ? { ...item, name: `${item.name} (edited)` }
            : item
        )
      );
      setLastAction(`Edited dataset "${dataset.name}".`);
      return;
    }

    setLastAction(`${action} triggered for "${dataset.name}".`);
  };

  const onAddDataset = () => {
    const nextDataset = createDataset(datasets.length);
    setDatasets((current) => [...current, nextDataset]);
    setSelectedId(nextDataset.id);
    setLastAction(`Added dataset "${nextDataset.name}".`);
  };

  return (
    <ThemeProvider>
      <Tabs
        aria-label="Datasets"
        selectedKey={selectedId}
        onSelectionChange={(key) => setSelectedId(String(key))}
        UNSAFE_style={{
          '--spectrum-tabs-selection-indicator-color': 'var(--energy-blue)',
          '--spectrum-tabs-emphasized-selection-indicator-color': 'var(--energy-blue)',
          '--spectrum-tabs-quiet-emphasized-selection-indicator-color': 'var(--energy-blue)',
          '--spectrum-tabs-item-gap': 'var(--spectrum-global-dimension-size-100)',
        }}
      >
        <div style={{ display: 'flex', alignItems: 'stretch', width: '100%' }}>
          <div style={{ flex: '0 0 auto', minWidth: 0 }}>
            <TabList>
              {datasets.map((dataset) => (
                <TabItem key={dataset.id} textValue={dataset.name}>
                  <ManagedTab
                    label={dataset.name}
                    isSelected={selectedId === dataset.id}
                    actions={ACTIONS}
                    onAction={(action) => onItemAction(action, dataset)}
                  />
                </TabItem>
              ))}
            </TabList>
          </div>

          <div
            style={{
              display: 'flex',
              flex: '1 1 auto',
              alignItems: 'center',
              justifyContent: 'flex-start',
              borderBottom:
                'var(--spectrum-alias-border-size-thick) solid var(--spectrum-global-color-gray-300)',
            }}
          >
            <ActionButton isQuiet aria-label="Add dataset" onPress={onAddDataset}>
              <Icon>
                <Add />
              </Icon>
            </ActionButton>
          </div>
        </div>

        <TabPanels>
          {datasets.map((dataset) => (
            <TabItem key={dataset.id}>
              <View paddingTop="size-150">
                <Text>
                  <strong>{dataset.name}</strong>
                </Text>
                <Text>{dataset.images} images</Text>
                <Text>Last action: {lastAction}</Text>
              </View>
            </TabItem>
          ))}
        </TabPanels>
      </Tabs>
    </ThemeProvider>
  );
}
```

## Props

### `ManagedTabProps`

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `label` | `string` | Yes | Visible tab label text. |
| `isSelected` | `boolean` | Yes | Whether this tab is currently active; controls menu affordance visibility and behavior. |
| `actions` | `ManagedTabAction[]` | Yes | Menu items shown when the tab is selected. |
| `onAction` | `(actionKey: string) => void` | Yes | Called when a menu action is selected. |
| `trailingContent` | `ReactNode` | No | Optional extra content rendered inside the tab label area. |

### `ManagedTabAction`

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `key` | `string` | Yes | Stable action identifier returned to `onAction`. |
| `label` | `string` | Yes | User-facing menu label. |

## Integration guidance

- Keep action handling in your application layer (state + API calls).
- Use `ManagedTab` inside `TabList` and keep `TabPanels` app-owned.
- Use stable keys for actions and tabs to avoid mismatched behavior after list updates.
