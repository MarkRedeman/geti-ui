# Inference Image

`buildInferenceImageInstance()` exposes OpenCV helpers for image preprocessing/postprocessing used in model workflows.

## Example

```ts
import { buildInferenceImageInstance } from '@geti/smart-tools';

const tool = await buildInferenceImageInstance();
const resized = tool.resize(imageData, 512, 512);
```

## Key API

- `resize(imageData, width, height)`
- `terminate()`
