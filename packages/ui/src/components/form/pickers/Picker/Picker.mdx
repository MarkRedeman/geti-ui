## Responsibility

`Picker` is a select-only dropdown that lets users choose a single value from a predefined list. Unlike `ComboBox`, the user cannot type a custom value. It wraps Spectrum's `Picker` and co-exports `Item` for option children.

## Design

Generic thin wrapper with a co-exported `Item`:

```tsx
export interface PickerProps<T extends object> extends SpectrumPickerProps<T> {}
export const Picker = <T extends object>(props: PickerProps<T>) => <SpectrumPicker {...props} />;
export { Item } from '@adobe/react-spectrum';
```

No CSS overrides. Full Spectrum API: `label`, `items`, `selectedKey`, `defaultSelectedKey`, `onSelectionChange`, `isDisabled`, `isRequired`, `placeholder`, `validationState`, `width`.

## Flow

Props in → forwarded to `SpectrumPicker` → Spectrum renders a button trigger + hidden `<select>` → user opens dropdown → selects an `Item` → `onSelectionChange` fires with the item key.

## Integration

- **Depends on**: `@adobe/react-spectrum` (`Picker`, `Item`, `SpectrumPickerProps`)
- **Usage**: `<Picker label="Size"><Item key="s">Small</Item><Item key="l">Large</Item></Picker>`
- **Contrast**: `ComboBox` (free-form text + suggestions) vs `Picker` (fixed options only)
