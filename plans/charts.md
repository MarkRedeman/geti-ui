# Charts

Let's add charts based on recharts.
The design criteria for this feature should be:

- Simple to use
- Good styles by default
- Minimal animations (possibly opt in)
- Performant for react 19 
- Allow for high interactivity by allow users to customize

The charts could come in two flavors:
- Ready to use full chart components that take in data
- Fully customizable chart components used via composition - more complex to setup but more freedom

Supported charts should be:

- Area Charts
- Bar Charts
- Line Charts
- Pie Charts
- Radar Charts
- Radial Charts
- Tooltips
