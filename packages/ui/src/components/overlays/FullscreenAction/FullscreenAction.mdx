## Responsibility

Provides a Geti-native composite pattern for "expand to fullscreen" actions: an icon button with a tooltip that, when pressed, opens a `fullscreenTakeover` dialog containing the caller's content with a close control. Used throughout the UI to promote panels, editors, and media viewers to a full-screen experience.

## Design

Fully custom composite (not a Spectrum pass-through). Composes Spectrum primitives directly from `@adobe/react-spectrum`:

- **Outer trigger** — `DialogTrigger type="fullscreenTakeover"` wraps everything.
- **Visible trigger element** — `TooltipTrigger` + `ActionButton` (renders `<FullScreen>` workflow icon, tooltip "Fullscreen"). `isQuiet` suppresses the button border.
- **Dialog content** — `Dialog` contains: `Heading` (from `title` prop), `Divider`, `ButtonGroup` (action buttons + close `ActionButton` with `<FullScreenExit>` icon inside `Content`), `Content ref={containerRef}` (the `children`).

**`actionButton` prop** — accepts either a `ReactNode` or a factory `(ref: RefObject<DOMRefValue<HTMLElement>>) => ReactNode`. The factory form gives callers a ref to the dialog's content container, enabling scenarios where the fullscreen content needs to measure or render into the container DOM node directly (e.g. canvas-based editors).

**`containerRef`** — `useRef<DOMRefValue<HTMLElement>>` — wired to Spectrum's `Content` component via the `ref` prop. Passed to `actionButton` factory if used.

## Flow

```
props { title, children, actionButton, isDisabled }
  → DialogTrigger type="fullscreenTakeover"
      → TooltipTrigger
          → ActionButton (FullScreen icon, tooltip "Fullscreen")
          → Tooltip "Fullscreen"
      → Dialog
          → Heading {title}
          → Divider
          → Content ref={containerRef}
              {children}
          → ButtonGroup
              {typeof actionButton === 'function'
                ? actionButton(containerRef)
                : actionButton}
              ActionButton (FullScreenExit icon, close)
```

## Integration

- Used in canvas editors, image viewers, and detail panels throughout the Geti application to support fullscreen review mode.
- The `actionButton` factory pattern is the mechanism for canvas-based components that need to render into the fullscreen container's DOM node.
- `FullScreen` and `FullScreenExit` icons are from `@spectrum-icons/workflow`.
- `DialogTrigger`, `Dialog`, `TooltipTrigger`, `ActionButton`, `Heading`, `Divider`, `Content`, `ButtonGroup` are all imported directly from `@adobe/react-spectrum`.
