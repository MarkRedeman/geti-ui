# Scatter chart (primitive)

Use Recharts `ScatterChart` with Geti primitives when you need bespoke scatter behavior.

## Props

Key Recharts props used on the `Scatter` primitive:

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `object[]` | required | Array of data points for this scatter series |
| `fill` | `string` | — | Point fill color |
| `name` | `string` | — | Series label shown in tooltip and legend |
| `shape` | `'circle' \| 'cross' \| 'diamond' \| 'square' \| 'triangle' \| 'star'` | `'circle'` | Marker shape |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |

`ZAxis` (bubble chart) props:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | — | Key used for bubble size |
| `range` | `[min, max]` | `[64, 64]` | Pixel area range for bubbles |
| `name` | `string` | — | Label shown in tooltip for the Z dimension |

## Variant — tooltip on

```tsx preview
import { Scatter, ScatterChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ui/charts/primitives';

const points = [
  { x: 0.1, y: 0.85 },
  { x: 0.3, y: 0.72 },
  { x: 0.5, y: 0.91 },
  { x: 0.7, y: 0.77 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={280}>
      <ScatterChart margin={{ top: 8, right: 16, bottom: 8, left: 0 }}>
        <GetiCartesianGrid />
        <GetiXAxis type="number" dataKey="x" name="False positive rate" />
        <GetiYAxis type="number" dataKey="y" name="True positive rate" />
        <GetiTooltip />
        <Scatter data={points} fill="var(--spectrum-global-color-blue-500)" isAnimationActive={false} />
      </ScatterChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — log/log axes

```tsx preview
import { Scatter, ScatterChart } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,

const models = [
  { x: 2e6, y: 1.2e9 },
  { x: 8e6, y: 4.6e9 },
  { x: 2.4e7, y: 1.1e10 },
  { x: 6.4e7, y: 3.0e10 },

    <GetiResponsiveContainer height={300}>
      <ScatterChart margin={{ top: 8, right: 16, bottom: 8, left: 0 }}>

```

## Variant — bubble scatter with Z axis

```tsx preview
import { Scatter, ScatterChart, ZAxis } from 'recharts';
import {
  GetiCartesianGrid,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ui/charts/primitives';

const points = [
  { x: 18, y: 0.72, z: 1200 },
  { x: 24, y: 0.81, z: 3400 },
  { x: 33, y: 0.87, z: 5200 },
  { x: 41, y: 0.90, z: 8600 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={300}>
      <ScatterChart margin={{ top: 8, right: 16, bottom: 8, left: 0 }}>
        <GetiCartesianGrid />
        <GetiXAxis type="number" dataKey="x" name="Latency (ms)" />
        <GetiYAxis type="number" dataKey="y" name="F1" />
        <ZAxis dataKey="z" range={[40, 400]} name="Samples" />
        <GetiTooltip />
        <Scatter data={points} fill="var(--spectrum-global-color-blue-500)" isAnimationActive={false} />
      </ScatterChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — model families (category + bubble size)

```tsx preview
import { Scatter, ScatterChart, ZAxis } from 'recharts';
import {
  GetiCartesianGrid,
  GetiLegend,
  GetiResponsiveContainer,
  GetiTooltip,
  GetiXAxis,
  GetiYAxis,
} from '@geti-ui/charts/primitives';

const yolo = [
  { params: 3.2e6, map: 0.41, gflops: 8.1 },
  { params: 1.1e7, map: 0.49, gflops: 28.7 },
  { params: 2.6e7, map: 0.54, gflops: 78.9 },
];

const unet = [
  { params: 7.8e6, map: 0.44, gflops: 21.5 },
  { params: 2.0e7, map: 0.51, gflops: 64.1 },
  { params: 3.9e7, map: 0.56, gflops: 118.3 },
];

const detr = [
  { params: 4.1e7, map: 0.52, gflops: 97.2 },
  { params: 6.8e7, map: 0.57, gflops: 162.4 },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={340}>
      <ScatterChart margin={{ top: 8, right: 16, bottom: 8, left: 8 }}>
        <GetiCartesianGrid />
        <GetiXAxis type="number" dataKey="params" scale="log" domain={[1e6, 'auto']} name="Parameters" />
        <GetiYAxis type="number" dataKey="map" name="M5 COCO mAP @ 50:95" />
        <ZAxis dataKey="gflops" range={[50, 460]} name="GFLOPs" />
        <GetiTooltip />
        <GetiLegend />

        <Scatter name="YOLO" data={yolo} fill="var(--spectrum-global-color-blue-500)" shape="circle" isAnimationActive={false} />
        <Scatter name="UNet" data={unet} fill="var(--spectrum-global-color-green-500)" shape="diamond" isAnimationActive={false} />
        <Scatter name="DETR" data={detr} fill="var(--spectrum-global-color-orange-500)" shape="triangle" isAnimationActive={false} />
      </ScatterChart>
    </GetiResponsiveContainer>
  );
}
```

Here, **model family** is represented by separate scatter series (driving legend + color), while bubble size encodes `gflops`.
