# ProgressCircle

`ProgressCircle` communicates progress with a circular indicator. Use it in compact spaces such as cards, tiles, and inline status regions where a horizontal bar is too wide.

```tsx
import { ProgressCircle } from '@geti-ai/ui';
```

- [React Spectrum ProgressCircle ↗](https://react-spectrum.adobe.com/react-spectrum/ProgressCircle.html)
- [Geti UI Theming](/components/theming)

## Examples

### Determinate

```tsx preview
import { ProgressCircle } from '@geti-ai/ui';

export default function App() {
  return <ProgressCircle aria-label="Training progress" value={64} />;
}
```

### Indeterminate

```tsx preview
import { ProgressCircle } from '@geti-ai/ui';

export default function App() {
  return <ProgressCircle aria-label="Loading" isIndeterminate />;
}
```

### Custom range

```tsx preview
import { ProgressCircle } from '@geti-ai/ui';

export default function App() {
  return <ProgressCircle aria-label="Epoch progress" minValue={0} maxValue={50} value={21} />;
}
```

## Responsibility

`ProgressCircle` is a thin wrapper around React Spectrum `ProgressCircle`.
It preserves Spectrum behavior and accessibility while keeping imports consistent in `@geti-ai/ui`.

## Variants and states

| State | Prop | Notes |
|---|---|---|
| Determinate | `value` | Displays numeric progress in min/max range |
| Indeterminate | `isIndeterminate` | Animated spinner-style progress |

## Props (deep dive)

### Value / state

- `value`: current progress value.
- `minValue` / `maxValue`: numeric range.
- `isIndeterminate`: animation without fixed percentage.

### Accessibility

- `aria-label` is required when no visible label is present.
- Prefer determinate mode if users can benefit from exact completion status.
