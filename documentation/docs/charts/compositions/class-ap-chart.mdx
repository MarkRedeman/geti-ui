# Class AP chart (composition)

`ClassAPChart` visualizes per-class average precision as sortable horizontal bars.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `ClassAPChartDatum[]` | required | Per-class AP rows |
| `metric` | `string` | `'AP50'` | Metric label in tooltip/legend |
| `sortDescending` | `boolean` | `true` | Sort by score descending |
| `showTooltip` | `boolean` | `true` | Show tooltip |
| `animate` | `boolean` | `false` | Opt-in animation |
| `xAxisProps` / `yAxisProps` | `XAxisProps` / `YAxisProps` | — | Axis passthrough props |

## Variant — AP50 per class

```tsx preview
import { ClassAPChart } from '@geti-ai/charts';

const data = [
  { className: 'person', score: 0.73 },
  { className: 'car', score: 0.81 },
  { className: 'traffic light', score: 0.42 },
  { className: 'bicycle', score: 0.66 },
  { className: 'truck', score: 0.58 },
  { className: 'bus', score: 0.61 },
  { className: 'dog', score: 0.49 },
  { className: 'cat', score: 0.53 },
];

export default function App() {
  return (
    <ClassAPChart
      data={data}
      metric="AP50"
      aria-label="Per-class AP50"
      height={420}
    />
  );
}
```
