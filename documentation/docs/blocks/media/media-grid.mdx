# Media grid

Use `MediaGrid` to render large media collections with virtualized scrolling, selection, and incremental loading.

`MediaGrid` follows a tell-don't-ask API:

- provide `totalItems`
- provide `getItemAt(index)`
- provide `renderItem(context)`

The grid does not assume how thumbnails should be rendered.

```tsx
import {
  MediaGrid,
  MediaGridItem,
  MediaGridThumbnailItem,
  MediaGridModeToggleButtons,
  DEFAULT_MEDIA_GRID_VIEW_MODES,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';
```

## Basic usage

```tsx preview
import { useMemo, useState } from 'react';
import { ThemeProvider } from '@geti-ai/ui';
import {
  MediaGrid,
  MediaGridThumbnailItem,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';

type Media = {
  id: string;
  name: string;
  thumbnailUrl: string;
};

const allMedia: Media[] = Array.from({ length: 200 }, (_, index) => ({
  id: `media-${index + 1}`,
  name: `Image ${index + 1}`,
  thumbnailUrl: `https://picsum.photos/seed/media-grid-${index + 1}/400/400`,
}));

export default function App() {
  const [loadedCount, setLoadedCount] = useState(60);
  const [selectedKeys, setSelectedKeys] = useState<MediaGridSelection>(new Set());

  const byIndex = useMemo(() => {
    const map = new Map<number, Media>();
    for (let i = 0; i < loadedCount; i += 1) {
      map.set(i, allMedia[i]);
    }
    return map;
  }, [loadedCount]);

  const renderItem = (ctx: MediaGridRenderContext<Media>) => (
    <MediaGridThumbnailItem
      src={ctx.item?.thumbnailUrl}
      alt={ctx.item?.name ?? 'Loading media'}
      isSelected={ctx.isSelected}
      isPlaceholder={ctx.isPlaceholder}
      onPress={ctx.onPress}
      onDoublePress={ctx.onDoublePress}
    />
  );

  return (
    <ThemeProvider>
      <div style={{ height: 520 }}>
        <MediaGrid
          totalItems={allMedia.length}
          getItemAt={(index) => byIndex.get(index)}
          hasNextPage={loadedCount < allMedia.length}
          isLoadingMore={false}
          onLoadMore={() => setLoadedCount((prev) => Math.min(prev + 40, allMedia.length))}
          selectionMode="multiple"
          selectedKeys={selectedKeys}
          onSelectionChange={setSelectedKeys}
          renderItem={renderItem}
        />
      </div>
    </ThemeProvider>
  );
}
```

## `MediaGrid` props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `totalItems` | `number` | Yes | Total number of items represented by the grid (loaded + not yet loaded). Used to keep virtualization and scrollbar size stable. |
| `getItemAt` | `(index: number) => T \| undefined` | Yes | Returns loaded item data for an absolute index, or `undefined` for unloaded slots. |
| `renderItem` | `(context: MediaGridRenderContext<T>) => ReactNode` | Yes | Renders each cell. Called for both loaded and placeholder entries. |
| `itemSize` | `number` | No | Square item size in px. Default: `200`. |
| `columns` | `number` | No | Forces a fixed number of columns. By default, columns are responsive based on container width. |
| `gap` | `number` | No | Grid gap in px (horizontal + vertical). Default: `8`. |
| `focusOnSelect` | `boolean` | No | When `true`, auto-scrolls to keep the selected item in view after selection changes. Default: `false`. |
| `selectionMode` | `'none' \| 'single' \| 'multiple'` | No | Selection behavior. Default: `'multiple'`. |
| `selectedKeys` | `Set<string> \| 'all'` | No | Controlled selection state. |
| `defaultSelectedKeys` | `Iterable<string>` | No | Initial uncontrolled selection state. |
| `onSelectionChange` | `(keys: Set<string> \| 'all') => void` | No | Called whenever selection changes. |
| `onItemPress` | `(context: MediaGridRenderContext<T>) => void` | No | Called when a loaded item is pressed. |
| `onItemDoublePress` | `(context: MediaGridRenderContext<T>) => void` | No | Called when a loaded item is double-pressed. |
| `hasNextPage` | `boolean` | No | Whether more data can be loaded. |
| `isLoadingMore` | `boolean` | No | Loading state for incremental fetches. |
| `onLoadMore` | `() => void` | No | Called when viewport approaches the end threshold. |
| `loadMoreThresholdRows` | `number` | No | Prefetch threshold expressed in rows from the end. Default: `2`. |
| `isLoading` | `boolean` | No | Initial/loading state for the whole grid. |
| `emptyState` | `ReactNode` | No | Custom empty-state UI when there are no items. |
| `loadingState` | `ReactNode` | No | Custom "loading more" row UI. |
| `className` | `string` | No | Optional class for the root container. |
| `style` | `CSSProperties` | No | Inline styles for the root container. |

## `MediaGridRenderContext` (for `renderItem`)

| Field | Type | Description |
| --- | --- | --- |
| `item` | `T \| undefined` | Loaded item data. `undefined` means this index has not been loaded yet. |
| `index` | `number` | Absolute item index in the virtualized collection. |
| `isPlaceholder` | `boolean` | `true` when the item at this index is not loaded yet. |
| `isSelected` | `boolean` | Current selection state for this item. |
| `selectionMode` | `'none' \| 'single' \| 'multiple'` | Active selection mode used by the grid. |
| `onPress` | `() => void` | Press handler wired by the grid (selection + optional callback). |
| `onDoublePress` | `() => void` | Double-press handler wired by the grid. |

## `MediaGridModeToggleButtons` props

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `options` | `Record<string, { label: string; itemSize: number }>` | No | Mode map. Defaults to `DEFAULT_MEDIA_GRID_VIEW_MODES`. |
| `value` | `string` | Yes | Selected mode key. |
| `onChange` | `(nextMode: string) => void` | Yes | Called when selected mode changes. |
| `isDisabled` | `boolean` | No | Disables the toggle buttons. |

## `DEFAULT_MEDIA_GRID_VIEW_MODES`

Default exported modes:

| Key | Label | itemSize |
| --- | --- | --- |
| `small` | `Small thumbnails` | `120` |
| `medium` | `Medium thumbnails` | `200` |
| `large` | `Large thumbnails` | `300` |

## Why `totalItems` + `getItemAt`?

`MediaGrid` computes scroll height from `totalItems`, so scrollbar size stays stable while pages are still loading.

`getItemAt(index)` lets you back the grid with an efficient indexed cache (for example, paginated query data mapped by absolute index) without allocating a full array of placeholders.

## Selection model

`MediaGrid` mirrors Spectrum-style controlled selection props.

By default, `MediaGrid` uses `selectionMode="multiple"`.

- `selectionMode`: `none | single | multiple` (default: `multiple`)
- `selectedKeys`
- `defaultSelectedKeys`
- `onSelectionChange`

## Item size and mode controls

`MediaGrid` only needs a numeric `itemSize`. Mode UI is separate.

Use `MediaGridModeToggleButtons` (optional helper) to drive your own mode state and convert mode → `itemSize`.

```tsx preview
import { useMemo, useState } from 'react';
import { ThemeProvider, Flex } from '@geti-ai/ui';
import {
  MediaGrid,
  MediaGridModeToggleButtons,
  MediaGridThumbnailItem,
  DEFAULT_MEDIA_GRID_VIEW_MODES,
  type MediaGridRenderContext,
} from '@geti-ai/blocks';

type Media = { id: string; name: string; thumbnailUrl: string };

const allMedia: Media[] = Array.from({ length: 120 }, (_, index) => ({
  id: `media-${index + 1}`,
  name: `Image ${index + 1}`,
  thumbnailUrl: `https://picsum.photos/seed/viewmode-${index + 1}/400/400`,
}));

const modes = {
  ...DEFAULT_MEDIA_GRID_VIEW_MODES,
  compact: { label: 'Compact', itemSize: 96 },
};

export default function App() {
  const [loadedCount, setLoadedCount] = useState(80);
  const [mode, setMode] = useState<'small' | 'medium' | 'large' | 'compact'>('compact');

  const byIndex = useMemo(() => {
    const map = new Map<number, Media>();
    for (let i = 0; i < loadedCount; i += 1) {
      map.set(i, allMedia[i]);
    }
    return map;
  }, [loadedCount]);

  const renderItem = (ctx: MediaGridRenderContext<Media>) => (
    <MediaGridThumbnailItem
      src={ctx.item?.thumbnailUrl}
      alt={ctx.item?.name ?? 'Loading media'}
      isPlaceholder={ctx.isPlaceholder}
      isSelected={ctx.isSelected}
      onPress={ctx.onPress}
      onDoublePress={ctx.onDoublePress}
    />
  );

  return (
    <ThemeProvider>
      <Flex direction="column" gap="size-150">
        <MediaGridModeToggleButtons options={modes} value={mode} onChange={(next) => setMode(next as typeof mode)} />
        <div style={{ height: 520 }}>
          <MediaGrid
            totalItems={allMedia.length}
            getItemAt={(index) => byIndex.get(index)}
            itemSize={modes[mode].itemSize}
            hasNextPage={loadedCount < allMedia.length}
            onLoadMore={() => setLoadedCount((prev) => Math.min(prev + 20, allMedia.length))}
            renderItem={renderItem}
          />
        </div>
      </Flex>
    </ThemeProvider>
  );
}
```

## Custom item composition

Use `renderItem` when you need custom content and overlays.

`MediaGridRenderContext` gives access to:

- loaded item (or placeholder state)
- index
- selection state
- `onPress` and `onDoublePress`

```tsx preview
import { useMemo, useState } from 'react';
import { ThemeProvider, ActionMenu, Checkbox, Flex, Text, View } from '@geti-ai/ui';
import { Item } from '@adobe/react-spectrum';
import {
  MediaGrid,
  MediaGridItem,
  MediaGridThumbnailItem,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';

type Media = { id: string; name: string; thumbnailUrl: string; score: number };

const allMedia: Media[] = Array.from({ length: 160 }, (_, index) => ({
  id: `media-${index + 1}`,
  name: `Asset ${index + 1}`,
  thumbnailUrl: `https://picsum.photos/seed/composed-${index + 1}/400/400`,
  score: Math.round((0.6 + (index % 40) / 100) * 100) / 100,
}));

export default function App() {
  const [loadedCount, setLoadedCount] = useState(70);
  const [selectedKeys, setSelectedKeys] = useState<MediaGridSelection>(new Set());

  const overlayTextStyle = {
    fontSize: '11px',
    color: 'var(--spectrum-global-color-gray-700)',
  } as const;

  const byIndex = useMemo(() => {
    const map = new Map<number, Media>();
    for (let i = 0; i < loadedCount; i += 1) {
      map.set(i, allMedia[i]);
    }
    return map;
  }, [loadedCount]);

  const renderCustomItem = (ctx: MediaGridRenderContext<Media>) => {
    if (ctx.isPlaceholder || !ctx.item) {
      return <MediaGridThumbnailItem isPlaceholder onPress={ctx.onPress} onDoublePress={ctx.onDoublePress} />;
    }
    
    const onCheckbox = (isPressed: boolean) => {
      const newSet = new Set(selectedKeys)
      if (!isPressed) {
        newSet.delete(ctx.item.id)
      } else{
        newSet.add(ctx.item.id)
      }
        setSelectedKeys(newSet)
    }

    return (
      <MediaGridItem
        isSelected={ctx.isSelected}
        onPress={ctx.onPress}
        onDoublePress={ctx.onDoublePress}
        topLeft={
          <Flex
            width="size-200"
            height="size-200"
            alignItems="center"
            justifyContent="center"
          >
            <Checkbox
              aria-label={`Select ${ctx.item.name}`}
              isSelected={ctx.isSelected}
              onChange={onCheckbox}
            />
          </Flex>
        }
        topRight={
          <ActionMenu
            isQuiet
            aria-label="Media actions"
            onAction={(key) => {
              console.log('action', key, 'for', ctx.item.id);
            }}
          >
            <Item key="annotate">Annotate</Item>
            <Item key="download">Download</Item>
            <Item key="delete">Delete</Item>
          </ActionMenu>
        }
        bottomLeft={
          <Text UNSAFE_style={overlayTextStyle}>{ctx.item.score}</Text>
        }
        bottomRight={
          <Text UNSAFE_style={overlayTextStyle}>{ctx.item.id}</Text>
        }
      >
        <img
          src={ctx.item.thumbnailUrl}
          alt={ctx.item.name}
          style={{ width: '100%', height: '100%', objectFit: 'cover', display: 'block' }}
        />
      </MediaGridItem>
    );
  };

  return (
    <ThemeProvider>
      <Flex direction="column" gap="size-100">
        <Text UNSAFE_style={{ fontSize: '12px', color: 'var(--spectrum-global-color-gray-700)' }}>
          Selected: {selectedKeys === 'all' ? 'all' : selectedKeys.size}
        </Text>
        <div style={{ height: 520 }}>
          <MediaGrid
            totalItems={allMedia.length}
            getItemAt={(index) => byIndex.get(index)}
            itemSize={180}
            hasNextPage={loadedCount < allMedia.length}
            onLoadMore={() => setLoadedCount((prev) => Math.min(prev + 30, allMedia.length))}
            selectionMode="multiple"
            selectedKeys={selectedKeys}
            onSelectionChange={setSelectedKeys}
            renderItem={renderCustomItem}
          />
        </div>
      </Flex>
    </ThemeProvider>
  );
}
```

## Varying thumbnail aspect ratios

This example uses source images with different intrinsic aspect ratios while keeping a stable square grid layout.

It also includes mode toggles to quickly validate behavior across item sizes.

```tsx preview
import { useMemo, useState } from 'react';
import { ThemeProvider, Flex } from '@geti-ai/ui';
import {
  MediaGrid,
  MediaGridModeToggleButtons,
  MediaGridThumbnailItem,
  DEFAULT_MEDIA_GRID_VIEW_MODES,
  type MediaGridRenderContext,
  type MediaGridSelection,
} from '@geti-ai/blocks';

type Media = {
  id: string;
  name: string;
  thumbnailUrl: string;
};

const sizePairs = [
  [640, 360],
  [360, 640],
  [1024, 768],
  [768, 1024],
  [1200, 400],
  [400, 1200],
  [500, 500],
];

const allMedia: Media[] = Array.from({ length: 180 }, (_, index) => {
  const [w, h] = sizePairs[index % sizePairs.length];

  return {
    id: `media-${index + 1}`,
    name: `Aspect ${w}x${h}`,
    thumbnailUrl: `https://picsum.photos/seed/aspect-${index + 1}/${w}/${h}`,
  };
});

const modes = {
  ...DEFAULT_MEDIA_GRID_VIEW_MODES,
  compact: { label: 'Compact', itemSize: 96 },
};

export default function App() {
  const [loadedCount, setLoadedCount] = useState(80);
  const [mode, setMode] = useState<'small' | 'medium' | 'large' | 'compact'>('medium');
  const [selectedKeys, setSelectedKeys] = useState<MediaGridSelection>(new Set());

  const byIndex = useMemo(() => {
    const map = new Map<number, Media>();

      src={ctx.item?.thumbnailUrl}

      isPlaceholder={ctx.isPlaceholder}
      onPress={ctx.onPress}
      onDoublePress={ctx.onDoublePress}

        <div style={{ height: 520 }}>

        </div>
      </Flex>
    </ThemeProvider>
  );
}
```

## Default components

- `MediaGridItem`: generic item shell with overlay slots (`topLeft`, `topRight`, `bottomLeft`, `bottomRight`)
- `MediaGridThumbnailItem`: default thumbnail item (`object-fit: cover`)
- `MediaGridModeToggleButtons`: optional mode control helper (`mode` → `itemSize`)
