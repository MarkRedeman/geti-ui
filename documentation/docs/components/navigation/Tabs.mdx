# Tabs

Tabs organise sibling content into labelled panels. Only one panel is visible at a time. Use tabs when the sections are parallel — for example Overview / Details / Settings — rather than hierarchical.

```tsx
import { TabItem as Item, TabList, TabPanels, Tabs } from '@geti-ai/ui';
```

- [React Spectrum Tabs ↗](https://react-spectrum.adobe.com/react-spectrum/Tabs.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default (horizontal)

```tsx preview
import { TabItem as Item, TabList, TabPanels, Tabs } from '@geti-ai/ui';
import { Content } from '@adobe/react-spectrum';

export default function App() {
  return (
    <Tabs aria-label="Project sections">
      <TabList>
        <Item key="overview">Overview</Item>
        <Item key="datasets">Datasets</Item>
        <Item key="models">Models</Item>
      </TabList>
      <TabPanels>
        <Item key="overview"><Content>Overview content.</Content></Item>
        <Item key="datasets"><Content>Datasets content.</Content></Item>
        <Item key="models"><Content>Models content.</Content></Item>
      </TabPanels>
    </Tabs>
  );
}
```

```tsx preview
  );
```


## Responsibility

`Tabs` is a thin wrapper around React Spectrum `Tabs`. Compose with `TabList`, `TabPanels`, and `TabItem` (re-exported as `Item`). Spectrum handles keyboard navigation (arrow keys), focus management, and ARIA roles automatically.

## Variants and states

| Prop | Options | Notes |
|---|---|---|
| `orientation` | `horizontal` (default) / `vertical` | Vertical suits sidebar-style settings pages |
| `isQuiet` | boolean | Removes the bottom border; use in panels with their own borders |
| `disabledKeys` | string[] | Disables specific tabs by key |
| `defaultSelectedKey` | string | Initial selected tab |
| `selectedKey` | string | Controlled selected tab |

## Props (deep dive)

### Content

- `children`: `TabList` and `TabPanels` must each contain matching `Item` elements with identical `key` values.

### Value / state

- `defaultSelectedKey`: initial tab (uncontrolled).
- `selectedKey` + `onSelectionChange`: controlled selection.
- `disabledKeys`: array of keys to disable.

### Visual

- `orientation`: `horizontal | vertical`.
- `isQuiet`: removes bottom line from the tab strip.
- `density`: `compact | regular` (Spectrum v3).

### Accessibility

- always provide `aria-label` on `Tabs` when there is no visible heading above the tab strip,
- tab keys must be unique strings within the same `Tabs` instance.
