# Parallel coordinates (composition)

`ParallelCoordinates` renders one polyline per row across multiple numeric axes (e.g. PCA components). It is useful for visually scanning clusters, outliers, and trends in high-dimensional embeddings.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `object[]` | required | Rows to render (one line per row) |
| `axes` | `[{ dataKey, label?, domain? }, ...]` | required | Ordered axis definitions (at least 2) |
| `colorBy` | `'left' \| 'right' \| string` | `'left'` | Color source axis: left, right, or an explicit axis dataKey (e.g. `'pc4'`) |
| `colorGradient` | `'theme' \| 'hue-spectrum' \| 'viridis' \| 'cividis' \| 'plasma' \| 'magma' \| 'inferno' \| 'turbo' \| 'distinct' \| string[]` | `'theme'` | Gradient preset or custom color stops |
| `legendLocation` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'bottom'` | Color legend placement |
| `colorDomain` | `[number, number]` | auto | Optional fixed min/max for the color-driving axis |
| `lineOpacity` | `number` | `0.22` | Base line opacity |
| `activeLineOpacity` | `number` | `0.95` | Opacity for active highlighted lines |
| `strokeWidth` | `number` | `1.2` | Polyline stroke width |
| `wideHitArea` | `boolean` | auto | Adds an invisible wider interaction path to improve hover/click |
| `hitAreaStrokeWidth` | `number` | auto | Width of the invisible interaction path when `wideHitArea` is enabled |
| `tickCount` | `number` | `4` | Tick labels per axis |
| `rowIdKey` | `string` | `'id'` | Field used as unique row id |
| `getRowId` | `(row, index) => string` | — | Custom id resolver |
| `onHoverRowChange` | `(row \| null, id \| null) => void` | — | Hover callback for linked UI (e.g. image preview) |
| `onSelectRowChange` | `(row, id) => void` | — | Click callback for pinned/selected line |
| `highlight` | `HighlightConfig` | — | Integrates with shared highlight system |
| `selection` | `SelectionConfig` | — | Drag-selection mode (`bounding-box`, `column`, `row`) and overlay style |
| `onSelectionChange` | `(selection \| null) => void` | — | Callback when drag selection completes/clears |
| `filterBySelection` | `boolean` | `false` | If true, render only lines fully inside selected region |

## Example datasets

```tsx preview
import { ParallelCoordinates } from '@geti-ai/charts';

function makePcaRows(rowCount: number, componentCount: number) {
  return Array.from({ length: rowCount }, (_, rowIndex) => {
    const row: Record<string, number | string> = {
      id: `img-${String(rowIndex + 1).padStart(4, '0')}`,
    };

    for (let component = 1; component <= componentCount; component += 1) {
      const base = -2.5 + (rowIndex / Math.max(1, rowCount - 1)) * 5;
      const wave = Math.sin((rowIndex + 1) * (component + 1) * 0.13) * 0.65;
      const trend = (component - 1) * 0.35;
      row[`pc${component}`] = Number((base + wave + trend).toFixed(3));
    }

    return row;
  });
}

const rows = makePcaRows(10, 8);
const axes = Array.from({ length: 8 }, (_, i) => ({
  dataKey: `pc${i + 1}`,
  label: `PC${i + 1}`,
})) as [
  { dataKey: string; label: string },
  { dataKey: string; label: string },
  ...{ dataKey: string; label: string }[]
];

export default function App() {
  return (
    <ParallelCoordinates
      data={rows}
      axes={axes}
      colorBy="pc4"
      colorGradient="hue-spectrum"
      lineOpacity={0.24}
      activeLineOpacity={0.95}
      highlight={{ enabled: true, interaction: { lineHover: true } }}
      aria-label="PCA parallel coordinates chart (8 components, 10 rows, hue spectrum)"
      height={460}
    />
  );
}
```

## Notes

- Use `onHoverRowChange` to drive linked panels (e.g. image preview, metadata table).
- Use `highlight` + `HighlightProvider` to share active row state across multiple charts.
- For dense datasets, start with lower `lineOpacity` (`0.08`–`0.2`) to reduce overplotting.
- Large row counts are primarily for overview density patterns; hover picking is best-effort and may become difficult at 1000+ lines.
- If hover targeting feels too difficult, set `wideHitArea` and tune `hitAreaStrokeWidth` for easier interaction while keeping visual strokes thin.
- `colorGradient="hue-spectrum"` provides a full rainbow-like mapping; for custom ramps pass color stops like `['#2b83ba', '#abdda4', '#fdae61', '#d7191c']`.

## Selection modes (drag to filter)

`ParallelCoordinates` supports drag-based selection overlays in three modes:

- `bounding-box`: free rectangle
- `column`: full-height vertical band (drag controls width)
- `row`: full-width horizontal band (drag controls height)

Double-click inside the plot area to clear selection.

### Selection example — user-controlled mode and filtering

```tsx preview
import { useMemo, useState } from 'react';
import type { SelectionMode, SelectionRect } from '@geti-ai/charts';
import { ParallelCoordinates } from '@geti-ai/charts';
import { ToggleButtons } from '@geti-ai/ui';

function makeRows(rowCount: number, componentCount: number) {
  return Array.from({ length: rowCount }, (_, rowIndex) => {
    const row: Record<string, number | string> = { id: `img-${rowIndex + 1}` };
    for (let c = 1; c <= componentCount; c += 1) {
      row[`pc${c}`] = Number((-2.6 + rowIndex * 0.003 + c * 0.22 + Math.sin((rowIndex + 1) * c * 0.05) * 0.8).toFixed(3));
    }
    return row;
  });
}

const allRows = makeRows(1000, 6);
const axes = Array.from({ length: 6 }, (_, i) => ({ dataKey: `pc${i + 1}`, label: `PC${i + 1}` })) as [
  { dataKey: string; label: string },
  { dataKey: string; label: string },
  ...{ dataKey: string; label: string }[]
];

  const [mode, setMode] = useState<SelectionMode>('bounding-box');
  const [selection, setSelection] = useState<SelectionRect | null>(null);

  const selectionConfig = useMemo(
    () => ({
      enabled: true,
      mode,
      overlayStyle: {
        fill: '#3b82f6',
        fillOpacity: 0.16,
        stroke: '#60a5fa',
        strokeWidth: 1.6,
      },
    }),
    [mode]
  );

  return (
    <div style={{ display: 'grid', gap: 12 }}>
      <ToggleButtons
        options={['bounding-box', 'row', 'column'] as SelectionMode[]}
        selectedOption={mode}
        onOptionChange={(next) => setMode(next as SelectionMode)}
      />

      <ParallelCoordinates
        data={allRows}
        axes={axes}
        colorBy="pc6"
```
