# Domain Images

SVG thumbnails representing each Geti annotation domain type.

```tsx
import { DetectionImg, ClassificationImg, SegmentationImg } from '@geti-ai/ui/assets/domains';
```

## Usage

Domain images render as inline SVG React components. Use them to visually represent task or project type:

```tsx
import { DetectionImg, ClassificationImg } from '@geti-ai/ui/assets/domains';

<DetectionImg width={48} height={48} />
<ClassificationImg width={48} height={48} />
```

## Import path

```ts
import { DetectionImg, SegmentationImg } from '@geti-ai/ui/assets/domains';

// Or via the combined namespace
import { Domains } from '@geti-ai/ui/assets';
const { DetectionImg } = Domains;
```
