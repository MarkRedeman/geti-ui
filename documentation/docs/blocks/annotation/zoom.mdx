# Zoom

An interactive pan-and-zoom viewport for 2D content such as images, videos, and annotation canvases. Supports scroll-wheel zoom, pinch-to-zoom, middle-mouse-button panning, `Ctrl`+drag panning, and programmatic zoom control.

```tsx
import { ZoomProvider, ZoomTransform, useZoom, useZoomActions } from '@geti-ai/blocks';
```

## Basic usage ã image with SVG overlays

Wrap your content in `ZoomProvider` and `ZoomTransform`. The `target` prop tells the viewport the natural dimensions of the content so it can compute fit-to-screen and pan boundaries.

Use the `--zoom-scale` CSS custom property exposed on the container to render scale-invariant SVG strokes that remain the same visual thickness regardless of zoom level.

```tsx preview
import { ZoomProvider, ZoomTransform, useZoom, useZoomActions } from '@geti-ai/blocks';
import { ThemeProvider, View, ActionButton, Text, Flex } from '@geti-ai/ui';

const IMAGE_WIDTH = 800;
const IMAGE_HEIGHT = 600;

// Define shapes with their bounding boxes for "focus on element" buttons
const SHAPES = [
  { name: 'Rectangle', rect: { x: 100, y: 80, width: 250, height: 180 }, color: '#00ff88' },
  { name: 'Circle', rect: { x: 420, y: 270, width: 160, height: 160 }, color: '#ff4444' },
  { name: 'Triangle', rect: { x: 280, y: 200, width: 140, height: 120 }, color: '#4488ff' },
];

function ZoomControls() {
  const { scale, minScale, maxZoomIn } = useZoom();
  const { zoomBy, fitToScreen, zoomTo } = useZoomActions();

  return (
    <Flex direction="row" gap="size-100" alignItems="center" wrap>
      <ActionButton
        onPress={() => zoomBy(-1)}
        isDisabled={scale <= minScale}
        aria-label="Zoom out"
      >
        -
      </ActionButton>
      <Text UNSAFE_style={{ minWidth: 48, textAlign: 'center', fontSize: 12, color: 'white' }}>
        {(scale * 100).toFixed(0)}%
      </Text>
      <ActionButton
        onPress={() => zoomBy(1)}
        isDisabled={scale >= maxZoomIn}
        aria-label="Zoom in"
      >
        +
      </ActionButton>
      <ActionButton onPress={fitToScreen} aria-label="Fit to screen">
        Fit
      </ActionButton>
      <View UNSAFE_style={{ width: 1, height: 20, background: '#444', margin: '0 4px' }} />
      {SHAPES.map((shape) => (
        <ActionButton
          key={shape.name}
          onPress={() => zoomTo(shape.rect, { padding: 20 })}
          aria-label={`Focus on ${shape.name}`}
        >
          <span style={{ color: shape.color, marginRight: 4 }}>ã</span>
          {shape.name}
        </ActionButton>
      ))}
    </Flex>
  );
}

function AnnotatedImage() {
  return (
    <div style={{ position: 'relative', width: IMAGE_WIDTH, height: IMAGE_HEIGHT }}>
      <img
        src="https://picsum.photos/800/600"
        width={IMAGE_WIDTH}
        height={IMAGE_HEIGHT}
        style={{ display: 'block', imageRendering: 'pixelated' }}
        alt="Sample"
      />
      <svg
        viewBox={`0 0 ${IMAGE_WIDTH} ${IMAGE_HEIGHT}`}
        width={IMAGE_WIDTH}
        height={IMAGE_HEIGHT}
        style={{ position: 'absolute', top: 0, left: 0, pointerEvents: 'none' }}
      >
        {/* Scale-invariant bounding box ã stroke stays 2px on screen */}
        <rect
          x={100} y={80} width={250} height={180}
          fill="none"
          stroke="#00ff88"
          strokeWidth="calc(2px / var(--zoom-scale, 1))"
        />
        {/* Scale-invariant circle */}
        <circle
          cx={500} cy={350} r={80}
          fill="none"
          stroke="#ff4444"
          strokeWidth="calc(2px / var(--zoom-scale, 1))"
          strokeDasharray="calc(8 / var(--zoom-scale, 1)) calc(4 / var(--zoom-scale, 1))"
        />
        {/* Scale-invariant polygon */}
        <polygon
          points="350,200 420,320 280,320"
          fill="none"
          stroke="#4488ff"
          strokeWidth="calc(2px / var(--zoom-scale, 1))"
        />
      </svg>
    </div>
  );
}

export default function App() {
  return (
    <ThemeProvider>
      <ZoomProvider target={{ width: IMAGE_WIDTH, height: IMAGE_HEIGHT }}>
        <View UNSAFE_style={{ display: 'flex', flexDirection: 'column', height: 450, background: '#1a1a2e' }}>
          <View UNSAFE_style={{ padding: 8 }}>
            <ZoomControls />
          </View>
          <View UNSAFE_style={{ flex: 1, minHeight: 0 }}>
            <ZoomTransform target={{ width: IMAGE_WIDTH, height: IMAGE_HEIGHT }} doubleClickMode="fitToScreen" fitPadding={16}>
              <AnnotatedImage />
            </ZoomTransform>
          </View>
        </View>
      </ZoomProvider>
    </ThemeProvider>
  );
}
```

## Non-interactive display grid

Set `interactive={false}` to disable all zoom and pan gestures. This is useful for thumbnail grids, read-only previews, or any context where you want fit-to-screen rendering without user interaction.

Each cell below uses a different image resolution and aspect ratio. The viewport automatically scales the content to fit.

```tsx preview
import { ZoomProvider, ZoomTransform } from '@geti-ai/blocks';
import { ThemeProvider } from '@geti-ai/ui';

const IMAGES = [
  { src: 'https://picsum.photos/800/600', width: 800, height: 600, label: '800 û 600 (4:3)' },
  { src: 'https://picsum.photos/1200/400', width: 1200, height: 400, label: '1200 û 400 (3:1)' },
  { src: 'https://picsum.photos/400/900', width: 400, height: 900, label: '400 û 900 (tall)' },
  { src: 'https://picsum.photos/600/600', width: 600, height: 600, label: '600 û 600 (1:1)' },
];

function CellHeader({ label }) {
  const buttonStyle = {
    fontSize: 10,
    padding: '2px 6px',
    background: '#333',
    color: '#ccc',
    border: '1px solid #555',
    borderRadius: 3,
    cursor: 'pointer',
  };

  return (
    <div style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      padding: '6px 10px',
      fontSize: 11,
      color: '#888',
      borderBottom: '1px solid #333',
    }}>
      <span>{label}</span>
    </div>
  );
}

function ImageCell({ src, width, height, label }) {
  return (
    <ZoomProvider target={{ width, height }}>
      <div style={{
        display: 'flex',
        flexDirection: 'column',
        height: '100%',
        background: '#1a1a2e',
        borderRadius: 6,
        overflow: 'hidden',
      }}>
        <CellHeader label={label} />
        <div style={{ flex: 1, minHeight: 0 }}>
          <ZoomTransform target={{ width, height }} interactive={false} >
            <img
              src={src}
              width={width}
              height={height}
              style={{ display: 'block', imageRendering: 'pixelated' }}
              alt={label}
            />
          </ZoomTransform>
        </div>
      </div>
    </ZoomProvider>
  );
}

export default function App() {
  return (
    <ThemeProvider>
      <div style={{
        display: 'grid',
        gridTemplateColumns: '1fr 1fr',
        gridTemplateRows: '200px 200px',
        gap: 8,
      }}>
        {IMAGES.map((img) => (
          <ImageCell key={img.label} {...img} />
        ))}
      </div>
    </ThemeProvider>
  );
}
```

## Interactions

| Input | Action |
| --- | --- |
| Scroll wheel | Zoom in/out, anchored at cursor position |
| Pinch gesture | Zoom in/out on touch devices, anchored at pinch midpoint |
| `Ctrl` + drag | Pan the viewport (cursor changes to grab/grabbing) |
| Middle mouse button + drag | Pan the viewport |
| Double-click | Reset to fit-to-screen (when `doubleClickMode="fitToScreen"`) |
| `+` / `-` buttons | Step zoom in/out (center-anchored) |
| Fit button | Reset to fit-to-screen with transition animation |

## Scale-invariant SVG rendering

`ZoomTransform` exposes a `--zoom-scale` CSS custom property on the outer container. Use it in `calc()` expressions to keep stroke widths, dash arrays, and other visual details constant on screen regardless of zoom level:

```css
/* Always appears as 2px on screen */
stroke-width: calc(2px / var(--zoom-scale, 1));

/* Consistent dash pattern */
stroke-dasharray: calc(8 / var(--zoom-scale, 1)) calc(4 / var(--zoom-scale, 1));
```

You can also read the scale in JavaScript via the `useZoom` hook:

```tsx
const { scale } = useZoom();
const anchorSize = ANCHOR_SIZE / scale;
```

## Programmatic zoom

The `useZoomActions` hook exposes imperative methods:

```tsx
const { fitToScreen, zoomBy, zoomTo } = useZoomActions();

// Zoom in by one step
zoomBy(1);

// Zoom out by two steps
zoomBy(-2);

// Zoom to fit a content-space rectangle in the viewport with 10px padding
zoomTo({ x: 100, y: 80, width: 250, height: 180 }, { padding: 10 });

// Zoom to fit a region without extra padding
zoomTo({ x: 0, y: 0, width: 400, height: 300 });

// Reset to fit-to-screen
fitToScreen();
```

## Props

### ZoomProvider

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | Yes | Content tree that needs zoom access. |
| `target` | `Size` | No | Content dimensions for pan boundary clamping. |

### ZoomTransform

| Prop | Type | Default | Required | Description |
| --- | --- | --- | --- | --- |
| `target` | `Size` | ã | Yes | Natural dimensions of the zoomable content. |
| `children` | `ReactNode` | ã | Yes | Content to render inside the viewport. |
| `interactive` | `boolean` | `true` | No | Whether zoom and pan interactions are enabled. Set to `false` for read-only display. |
| `doubleClickMode` | `'fitToScreen' \| 'none'` | `'none'` | No | Action on double-click. `'fitToScreen'` resets to fit-to-screen with animation. |
| `zoomInMultiplier` | `number` | `10` | No | Max zoom = initial fit-to-screen scale multiplied by this value. |
| `zoomOutMultiplier` | `number` | `0.5` | No | Min zoom = initial fit-to-screen scale multiplied by this value. Set to `1` to prevent zooming out past fit-to-screen. |
| `fitPadding` | `number` | `0` | No | Pixel padding around content when fitting to screen. `0` means content fills the container edge-to-edge. |

### useZoom

Returns the combined zoom state:

| Field | Type | Description |
| --- | --- | --- |
| `scale` | `number` | Current zoom scale. |
| `translate` | `Point` | Current pan offset `{ x, y }` in pixels. |
| `initialCoordinates` | `{ scale, x, y }` | Fit-to-screen baseline values. |
| `minScale` | `number` | Lower zoom scale bound. |
| `maxZoomIn` | `number` | Upper zoom scale bound. |

### useZoomActions

Returns imperative zoom control methods:

| Method | Signature | Description |
| --- | --- | --- |
| `fitToScreen` | `() => void` | Reset to initial fit-to-screen position. |
| `zoomBy` | `(step: number) => void` | Increment/decrement zoom by discrete steps. `1` = zoom in, `-1` = zoom out. |
| `zoomTo` | `(viewport: Rect, options?: ZoomToOptions) => void` | Zoom and pan so that the given content-space rectangle is visible in the viewport, centered and scaled to fit with optional padding. |

## Types

```ts
type Point = { x: number; y: number };
type Size = { width: number; height: number };
type Rect = { x: number; y: number; width: number; height: number };

type ZoomToOptions = {
  /** Extra padding (in viewport pixels) around the target rectangle. Default: 0 */
  padding?: number;
};

type ZoomConfig = {
  initialCoordinates: Point & { scale: number };
  minScale: number;
  maxZoomIn: number;
};

type ZoomTransformState = {
  scale: number;
  translate: Point;
};
```
