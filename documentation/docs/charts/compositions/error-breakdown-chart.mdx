# Error breakdown chart (composition)

`ErrorBreakdownChart` visualizes model error categories (false positives, false negatives, localization errors, classification errors).

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Record<string, unknown>[]` | — | Array of data objects, one per model/run |
| `series` | `BarSeriesConfig[]` | — | Series definitions; each entry has `dataKey` and `name` for an error category |
| `xAxisKey` | `string` | `'label'` | Key in each data object for the X (category) axis |
| `height` | `number` | `300` | Chart height in pixels |
| `stacked` | `boolean` | `true` | Stack error categories; set `false` for grouped bars |
| `showGrid` | `boolean` | `true` | Show background grid lines |
| `showTooltip` | `boolean` | `true` | Show hover tooltip |
| `showLegend` | `boolean` | `true` | Show legend below chart |
| `animate` | `boolean` | `false` | Enable entry animation |
| `xAxisProps` | `XAxisProps` | — | Extra props forwarded to the Recharts `XAxis` |
| `yAxisProps` | `YAxisProps` | — | Extra props forwarded to the Recharts `YAxis` |

## Variant — stacked breakdown by model

```tsx preview
import { ErrorBreakdownChart } from '@geti-ai/charts';

const data = [
  { label: 'YOLOv8-n', fp: 124, fn: 96, loc: 71, cls: 39 },
  { label: 'YOLOv8-s', fp: 108, fn: 82, loc: 55, cls: 31 },
  { label: 'RT-DETR', fp: 93, fn: 75, loc: 48, cls: 27 },
];

export default function App() {
  return (
    <ErrorBreakdownChart
      data={data}
      xAxisKey="label"
      series={[
        { dataKey: 'fp', name: 'False positives' },
        { dataKey: 'fn', name: 'False negatives' },
        { dataKey: 'loc', name: 'Localization errors' },
        { dataKey: 'cls', name: 'Classification errors' },
      ]}
      aria-label="Stacked error breakdown by model"
      height={340}
    />
  );
}
```

## Variant — grouped comparison by run

```tsx preview
import { ErrorBreakdownChart } from '@geti-ai/charts';

const data = [
  { label: 'Run A', fp: 140, fn: 112, loc: 82, cls: 44 },
  { label: 'Run B', fp: 126, fn: 103, loc: 77, cls: 40 },
  { label: 'Run C', fp: 110, fn: 91, loc: 64, cls: 34 },
];

export default function App() {
  return (
    <ErrorBreakdownChart
      data={data}
      stacked={false}
      xAxisKey="label"
      series={[
        { dataKey: 'fp', name: 'False positives' },
        { dataKey: 'fn', name: 'False negatives' },
        { dataKey: 'loc', name: 'Localization errors' },
        { dataKey: 'cls', name: 'Classification errors' },
      ]}
      aria-label="Grouped error breakdown by run"
      height={340}
    />
  );
}
```
