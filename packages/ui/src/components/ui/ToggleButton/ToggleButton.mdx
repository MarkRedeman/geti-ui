## Responsibility

`ToggleButton` is a single on/off toggle control. It wraps Spectrum's `ToggleButton` with no prop changes, serving purely as the Geti-library entry point so consumers never import from `@adobe/react-spectrum` directly.

<Stories />
---

## Design

### Minimal Spectrum wrapper

```tsx
export interface ToggleButtonProps extends SpectrumToggleButtonProps {}
export const ToggleButton = (props: ToggleButtonProps) => <SpectrumToggleButton {...props} />;
```

All Spectrum props pass through unchanged:
- `isSelected` / `defaultSelected` — controlled/uncontrolled selected state
- `onChange` — fires with the new boolean selected value
- `isEmphasized` — uses the accent colour when selected
- `isQuiet` — minimal border/background style
- `isDisabled`, `aria-label`, `children`, etc.

No CSS module, no Geti-specific props, no prop interception.

---

## Flow

Pure presentational pass-through. Spectrum manages the toggle state internally when uncontrolled, or delegates to the caller when `isSelected` is provided.

---

## Integration

- **Depends on**: `@adobe/react-spectrum` (`ToggleButton`, `SpectrumToggleButtonProps`).
- **Distinct from `ToggleButtons`**: `ToggleButton` is a standalone single control with its own on/off state. `ToggleButtons` is a multi-option exclusive group built from `Button` instances.
- **Consumed by**: individual toolbar toggles (e.g. bold/italic, show/hide panel) where a single binary state is needed.
