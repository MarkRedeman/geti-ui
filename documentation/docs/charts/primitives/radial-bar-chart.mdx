# RadialBar chart (primitive)

Use Recharts `RadialBarChart` primitives when you need full control over radial starts/ends, tracks, and labels.

## Props

Key Recharts props used on the `RadialBarChart` and `RadialBar` primitives:

| Prop | Type | Default | Description |
|---|---|---|---|
| `dataKey` | `string` | required | Key in each data object to plot on the `RadialBar` |
| `startAngle` | `number` | `90` | Start angle in degrees for the radial sweep |
| `endAngle` | `number` | `-270` | End angle in degrees; use `450` for clockwise direction |
| `innerRadius` | `number \| string` | — | Inner bound of the radial bar rings |
| `outerRadius` | `number \| string` | `'80%'` | Outer bound of the radial bar rings |
| `background` | `boolean \| object` | `false` | Render a background track behind each bar |
| `cornerRadius` | `number` | `0` | Rounded corner radius on bar ends |
| `isAnimationActive` | `boolean` | `true` | Disable for static/SSR renders |

## Variant — default

```tsx preview
import { PolarAngleAxis, RadialBar, RadialBarChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ui/charts/primitives';

const data = [
  { name: 'Precision', value: 88, fill: 'var(--spectrum-global-color-blue-500)' },
  { name: 'Recall', value: 84, fill: 'var(--spectrum-global-color-green-500)' },
  { name: 'F1', value: 86, fill: 'var(--spectrum-global-color-yellow-500)' },
  { name: 'Stability', value: 74, fill: 'var(--spectrum-global-color-purple-500)' },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={360}>
      <RadialBarChart data={data} startAngle={90} endAngle={-270} innerRadius="20%" outerRadius="90%">
        <PolarAngleAxis type="number" domain={[0, 100]} tick={false} />
        <RadialBar dataKey="value" background cornerRadius={6} isAnimationActive={false} />
        <GetiTooltip />
        <GetiLegend />
      </RadialBarChart>
    </GetiResponsiveContainer>
  );
}
```

## Variant — train / validation / test palette

```tsx preview
import { PolarAngleAxis, RadialBar, RadialBarChart } from 'recharts';
import { GetiLegend, GetiResponsiveContainer, GetiTooltip } from '@geti-ui/charts/primitives';
import { getDatasetSubsetColor } from '@geti-ui/charts';

const data = [
  { name: 'Train', value: 91, fill: getDatasetSubsetColor('train') },
  { name: 'Validation', value: 86, fill: getDatasetSubsetColor('validation') },
  { name: 'Test', value: 84, fill: getDatasetSubsetColor('test') },
];

export default function App() {
  return (
    <GetiResponsiveContainer height={340}>
      <RadialBarChart data={data} startAngle={90} endAngle={-270} innerRadius="28%" outerRadius="90%">
        <PolarAngleAxis type="number" domain={[0, 100]} tick={false} />
        <RadialBar dataKey="value" background cornerRadius={6} isAnimationActive={false} />
        <GetiTooltip />
        <GetiLegend />
      </RadialBarChart>
    </GetiResponsiveContainer>
  );
}
```
