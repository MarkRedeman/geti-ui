# Footer

`Footer` defines the bottom region of a dialog, typically for action controls.

```tsx
import { Footer } from '@geti-ui/ui';
```

- [React Spectrum Footer ↗](https://react-spectrum.adobe.com/react-spectrum/Dialog.html)
- [Geti UI Theming](/components/theming)

## Examples

### Dialog footer actions

```tsx preview
import { Button, ButtonGroup, Content, Dialog, DialogTrigger, Divider, Footer, Heading } from '@geti-ui/ui';

export default function App() {
  return (
    <DialogTrigger>
      <Button variant="accent">Open dialog</Button>
      {(close) => (
        <Dialog>
          <Heading>Unsaved changes</Heading>
          <Divider />
          <Content>You have unsaved edits. Do you want to leave without saving?</Content>
          <Footer>
            <ButtonGroup>
              <Button variant="secondary" onPress={close}>Cancel</Button>
              <Button variant="negative">Discard</Button>
            </ButtonGroup>
          </Footer>
        </Dialog>
      )}
    </DialogTrigger>
  );
}
```

## Responsibility

`Footer` is a thin wrapper around React Spectrum `Footer`. It provides a consistent import path for dialog footer composition in `@geti-ui/ui`.
