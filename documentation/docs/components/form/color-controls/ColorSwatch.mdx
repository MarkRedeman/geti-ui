# ColorSwatch

`ColorSwatch` renders a small filled square that previews a color value. Use it anywhere a color needs a visual representation — in pickers, palettes, or alongside labels.

```tsx
import { ColorSwatch } from '@geti-ai/ui';
```

- [React Spectrum ColorSwatch ↗](https://react-spectrum.adobe.com/react-spectrum/ColorSwatch.html)
- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { ColorSwatch } from '@geti-ai/ui';

export default function App() {
  return <ColorSwatch color="#ff0000" aria-label="Red" />;
}
```

### Sizes

```tsx preview
import { ColorSwatch, Flex } from '@geti-ai/ui';

export default function App() {
  return (
    <Flex gap="size-200">
      <ColorSwatch size="S" color="#ff0000" aria-label="Small red" />
      <ColorSwatch size="M" color="#00ff00" aria-label="Medium green" />
      <ColorSwatch size="L" color="#0000ff" aria-label="Large blue" />
    </Flex>
  );
}
```

### Rounding variants

```tsx preview
import { ColorSwatch, Flex } from '@geti-ai/ui';

export default function App() {
  return (
    <Flex gap="size-200">
      <ColorSwatch rounding="none" color="#ff0000" aria-label="Square" />
      <ColorSwatch rounding="default" color="#00ff00" aria-label="Rounded" />
      <ColorSwatch rounding="full" color="#0000ff" aria-label="Circle" />
    </Flex>
  );
}
```

## Responsibility

`ColorSwatch` is a direct pass-through of React Spectrum's `ColorSwatch`. It accepts any CSS or Spectrum color string.

## Variants and states

| Prop | Values | Notes |
|---|---|---|
| `size` | `"S"`, `"M"`, `"L"` | Controls physical dimensions |
| `rounding` | `"none"`, `"default"`, `"full"` | Corner radius |

## Props (deep dive)

### Content

- `color`: any valid CSS color string, Spectrum color string, or `Color` object.
- `aria-label`: **required** — the swatch is a visual-only element and needs an accessible name.

### Visual

- `size`: `"S" | "M" | "L"` — swatch dimensions.
- `rounding`: `"none" | "default" | "full"` — corner radius.

### Accessibility

- Always provide `aria-label` describing the color (e.g. `"Red"`).
- When used inside `ColorSwatchPicker`, selection state is announced automatically.
