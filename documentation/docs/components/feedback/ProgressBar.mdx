# ProgressBar

`ProgressBar` shows task progress in a horizontal track. Use it when users need a quick percentage-style signal for long-running operations.

```tsx
import { ProgressBar } from '@geti-ui/ui';
```

- [React Spectrum ProgressBar ↗](https://react-spectrum.adobe.com/react-spectrum/ProgressBar.html)
- [Geti UI Theming](/components/theming)

## Examples

### Determinate progress

```tsx preview
import { ProgressBar } from '@geti-ui/ui';

export default function App() {
  return <ProgressBar label="Uploading" value={45} />;
}
```

### Indeterminate

```tsx preview
import { ProgressBar } from '@geti-ui/ui';

export default function App() {
  return <ProgressBar label="Processing" isIndeterminate />;
}
```

### Custom min/max

```tsx preview
import { ProgressBar } from '@geti-ui/ui';

export default function App() {
  return <ProgressBar label="Batch" minValue={0} maxValue={20} value={8} />;
}
```

## Responsibility

`ProgressBar` is a pass-through wrapper around React Spectrum `ProgressBar`.
It keeps the same behavior while standardizing imports through `@geti-ui/ui`.

## Variants and states

| State | Prop | Notes |
|---|---|---|
| Determinate | `value` | Shows concrete progress within min/max |
| Indeterminate | `isIndeterminate` | Shows continuous animation without percent |
| Disabled | `isDisabled` | Non-interactive visual state |

## Props (deep dive)

### Content

- `label`: visible progress label.
- `formatOptions`: Intl number formatting options for displayed value.

### Value / state

- `value`: current progress value.
- `minValue` / `maxValue`: progress range.
- `isIndeterminate`: animated non-determinate state.

### Accessibility

- Always provide `label` so screen readers announce purpose, not only value.
- Use determinate mode whenever percentage can be computed.
