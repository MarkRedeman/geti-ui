## Responsibility

`TagGroup` is an interactive collection of tags used for categorizing, filtering, or labeling items. It provides a focusable list that supports keyboard navigation, single or multiple selection, and individual item removal.

### When to use TagGroup
- **Interactive Labels**: Displaying keywords, categories, or metadata that users can interact with.
- **Selection**: When users need to select one or more tags from a group (e.g., filtering a list).
- **Removal**: When users need to dismiss or delete tags (e.g., removing applied filters).
- **Dynamic Collections**: Managing a list of items that can grow or shrink based on user action.

### Comparison with other components
| Scenario | Use instead |
| :--- | :--- |
| Static, non-interactive status label (e.g. "Beta") | `Badge` |
| Read-only label paired with a form field | `Label` |
| Inline status indicators (error/warning) | `StatusLight` |

### Usage Rules
- **Always provide a label**: Use the `label` prop, or `aria-label` if no visual label is needed.
- **Removal requires `onRemove`**: The remove button only appears when the `onRemove` handler is provided.
- **`maxRows` for overflow**: Use `maxRows` to cap visible rows; an auto-generated "Show more" button will handle the overflow.

## Variants

### Selection Modes
Supports `none` (default), `single`, and `multiple` selection modes.

### Removable Tags
Tags become removable when the `onRemove` prop is passed to the `TagGroup`.

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `children` | `ItemElement \| ItemRenderer` | — | The items to display. Typically `<Item>` nodes. |
| `label` | `ReactNode` | — | The visual label for the tag group. |
| `onRemove` | `(keys: Set<Key>) => void` | — | Handler called when a tag is removed. |
| `selectionMode` | `'none' \| 'single' \| 'multiple'` | `'none'` | The selection mode for the group. |
| `maxRows` | `number` | — | The maximum number of rows to display before truncating. |
| `items` | `Iterable<T>` | — | The list of items for dynamic collections. |

## Accessibility

### Roles and Navigation
- `TagGroup` renders as a **`role="grid"`**. Each tag is a **`role="row"`** containing a **`role="gridcell"`**.
- **Keyboard Navigation**:
    - `Arrow keys`: Move focus between tags.
    - `Space / Enter`: Select/deselect the focused tag.
    - `Backspace / Delete`: Remove the focused tag (if `onRemove` is provided).
    - `Home / End`: Jump to the first or last tag.

### Requirements
- **Accessible Name**: A `label` or `aria-label` is mandatory for the grid landmark.
- **Text Value**: Provide `textValue` on an `<Item>` if its children are not plain text (e.g., if it contains an icon).
