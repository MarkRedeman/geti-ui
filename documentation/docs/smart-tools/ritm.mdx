# RITM (ONNX)

RITM is an iterative click-based segmentation model.

## Imports

```ts
import { buildRITMInstance, RITMModels, RITM_TEMPLATE_SIZE } from '@geti-ui/smart-tools/ritm';
```

## Example

```ts
const ritm = await buildRITMInstance();
await ritm.load();

ritm.loadImage(imageData);
const shape = await ritm.execute(
  { x: 0, y: 0, width: imageData.width, height: imageData.height },
  [{ x: 100, y: 120, positive: true }],
  'polygon'
);
```
