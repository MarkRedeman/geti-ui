# ColorThumb

`ColorThumb` is a small square that displays a solid color preview. It is a lightweight Geti-native component (not from React Spectrum) built on `View`. Use it wherever a compact color chip is needed ŌĆö for example, next to a label name or in a list row.

```tsx
import { ColorThumb } from '@geti-ai/ui';
```

- [Geti UI Theming](/components/theming)

## Examples

### Default

```tsx preview
import { ColorThumb } from '@geti-ai/ui';

export default function App() {
  return <ColorThumb color="#ff0000" size={20} />;
}
```

### Size variants

```tsx preview
import { ColorThumb, Flex } from '@geti-ai/ui';

export default function App() {
  return (
    <Flex gap="size-200" alignItems="center">
      <ColorThumb color="#ff0000" size={10} />
      <ColorThumb color="#00ff00" size={20} />
      <ColorThumb color="#0000ff" size={40} />
    </Flex>
  );
}
```

### Using CSS color strings

```tsx preview
import { ColorThumb, Flex } from '@geti-ai/ui';

export default function App() {
  return (
    <Flex gap="size-200" alignItems="center">
      <ColorThumb color="rgba(255,165,0,0.8)" size={20} />
      <ColorThumb color="hsl(270, 60%, 50%)" size={20} />
      <ColorThumb color="#cccccc" size={20} />
    </Flex>
  );
}
```

## Responsibility

`ColorThumb` is a Geti-native component, not from React Spectrum. It renders a `View` with a fixed square size and a CSS custom property (`--color-thumb-bg`) to set the background color. Static layout (border-radius, border) is owned by `ColorThumb.module.css`.

Key design choice: only the CSS variable is dynamic ŌĆö the CSS module controls all visual structure.

## Variants and states

| Prop | Type | Default | Notes |
|---|---|---|---|
| `color` | `string` | **required** | Any valid CSS color string |
| `size` | `number` | `10` | Width and height in pixels |

## Props (deep dive)

### Content

- `color` (**required**): any valid CSS color string (hex, rgb, rgba, hsl, etc.).

### Visual

- `size`: width and height of the square in pixels. Defaults to `10`.
- `UNSAFE_className`: additional CSS class to merge with the module styles.
- `UNSAFE_style`: additional inline styles.

### Accessibility

- `ColorThumb` is purely decorative. Wrap it with an accessible label in context (e.g., the label text next to it).
